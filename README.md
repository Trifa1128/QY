# 七印解封（26）：诺亚方舟是真的

2020/09/25
因为诺亚方舟在西方文化中占有比较重的分量，所以我们在讲中国人神共通这件事情之前还是先把诺亚方舟这个事情说了。

诺亚方舟的故事来自于圣经《创世纪》，大意是说耶和华见地上人类道德败坏，于是计划用洪水消灭恶人，然后他发现人类之中有一位叫做诺亚的好人，于是他就指示诺亚建造一艘方舟，并带着他的妻子、儿子与媳妇以及牲畜与鸟类等动物雌雄一对对的带上方舟。我们从七印书卷看到，这个诺亚方舟是真的，诺亚这一家人也是真的，很多牲畜和动物被带到方舟上也是真的，但是用洪水毁灭人类不是耶和华的主意。

地上人类涉及到东西方佛家道家两大体系，所以涉及整个地球的变化不是哪一家单独就能决定，以洪水毁灭上一次人类文明是当初制定七印书卷的那些神定下的。耶和华知道人类社会要发生的事情，而对于下一次人类文明要做的事情耶和华也早有计划。我们在前面关于本次人类文明时期犹太人的历史已经提到，本次人类文明犹太人是被耶和华拣选来配合耶稣留下一段关于救赎的历史的，是要为创世主预备道路的。所以在看到人类要被大洪水灭世，耶和华就需要留下一些犹太人为即将开始的新的文明历史准备，于是他就拣选了诺亚作为留下犹太人的关键。当然诺亚能被耶和华拣选也是因为诺亚本身对神的信达到了那个程度。耶和华是以梦境的方式启示诺亚的，诺亚在梦中看到有神告诉他不久人类就要被洪水毁灭，要他造一艘巨型的船，其中要有一层是给动物准备的，诺亚醒来后相信这个梦是真的，于是他就按梦中所见到的那艘船开始了庞大的造船计划。

诺亚一家那个时候是住城外，在当时属于比较有钱的人家。这个城按今天地理位置来看应该在土耳其迪亚巴克尔和尚勒乌尔法之间。当地有一些人也相信了诺亚的话，于是加入到方舟建造中来，但人手还是远远不够，于是诺亚就出钱请了很多人来造方舟。这个方舟不是在水里造的，它是在陆地上一块开阔地方造的。方舟外型上就是长方形的，船底也是平的，大概有现在一个半的标准足球场那么大，共五层，每上一层就稍微比下一层小一些，每一层也是方的。其中第一层是储存间，第二层是给动物准备的，第三层是用来做饭吃饭活动等等之类的，第四层和第五层是用来住人的。方舟大约四个月就造好后，神通过梦境指示诺亚马上让人搬到方舟上，因为洪水马上要来了，于是那些不相信的工人拿了工钱就回家了，而那些相信的就立即把自己的家人、家里的家禽家畜之类的，等等都带到船上，远远近近也有些人也都赶过来，然后第二天很多动物就从树林里走了出来，就象人很懂事一样上到第二层神在梦境中吩咐诺亚为它们准备的地方，其实这些动物在方舟造好前的几天就已经陆陆续续在神的指引下来到外面的树林里等着，只是那时候人们没发现。方舟造好两天后，灭世的大洪水就来了，当时上到诺亚方舟上的人大概有三千多。

人总以为再大的天灾也会有预警，人也能提前做好准备逃生，其实人类毁灭的方式很多时候都不是当时人所能够提前预警的，也不是慢慢来的。故事说这场灭世洪水是因为全球连下了四十天的暴雨导致的，我们没看到这个情况，这个灭世洪水是在地球上突然出现的，人根本就来不及逃生。我们从七印书卷看到，当灭世的时辰一到，全球各地就来了约176位神，其中约100位是属于佛家体系的，76位是属于道家体系的，他们按不同方位分布在全球各大海洋湖泊的上空。只见这些神打出手印，然后在他们手印的下方就出现了一个金色的圆形的字，他们转动这个字，这个圆形的字就象一把钥匙一样，下面的海水或者湖水就会分开一个圆洞直达海底或湖底，然后地下就裂开一个口。当这个口一裂开，那些神就立即飞走了，然后就看到全世界各地从那个口里出来无数的水，就象是突然从土里冒出来一样一下子就把地球淹了，这个速度之快，那些地面上的人根本来不及自救。

这些水的来源我们也看了一下，是众神从南瞻部洲所在的那个南海里把差不多一滴的水珠放进了地球。

诺亚方舟被那些突然出现的大洪水托到水面上，耶和华用他的神力给这个方舟下了一个罩把方舟保护起来，以免它因为颠簸或者漏水而翻了，这个罩诺亚他们看不到。如果不是耶和华的保护，这艘方舟早就七零八碎了。圣经故事说洪水淹没了世界最高的山，我们也没看到这个情况，世界上不少高山并没有被淹没，但因为洪水来的太快，世人根本就来不及往山上跑，只有那些原本住在山上的，或者听信了神的启示而提前跑到山上的才活下来。诺亚方舟后来飘到了亚拉腊山，是停靠在这座山半山腰往上一些的地方，因为半山腰往上就没被水淹了，并非圣经故事描述的山顶上。那里当时山上也是森林，于是船上的人和动物全部都离开船去到山上等待洪水退去，他们也在山上盖房子住下，这一等也象中国人一样差不多一年。而当洪水慢慢退去时，那些退去的洪水也把诺亚他们造的那艘方舟慢慢拖到海里沉下去了，所以其实这个方舟作为一个整体目前在地球已经不存在了。于是，本次人类文明开始后，犹太人就首先出现在土耳其。圣经关于洪水、方舟的故事是真的，但里面很多情节就有问题，甚至说诺亚活了多少百岁，我们看到他是一个普通人，没超过普通人活的岁数。

当时不仅诺亚他们，整个地球上不同民族国家的人们都得到了他们的神用各种方式启示他们的人，告诉他们即将到来的灭世洪水，有人相信，但绝大部分都是不相信的，那些相信的就往高山上躲避，那些高山上幸存下来的人就成为了本次人类文明各民族的祖先。而在中国也有很多神带来的启示，也有很多地上的修道人出来告诉当时人们这个真相，那些相信的就往青藏高原那里的高山跑，于是那些高山上幸存下来的人就成为了本次人类文明中国人的祖先。而且在大洪水来之前，地球上那些没有被洪水淹到的高山上本来就有很多动物，当大洪水出现后，那些本来在山脚的一些动物也都跑到山上来了，所以，本次人类文明时期地球上的动物大部分都不是从诺亚方舟那里来的。

#  七印解封（27）：打开人神共通之门

2020/09/28
人类历史关于大洪水灭世的传说只说了地球上的人类文明被毁灭，其实每一次涉及整个地球的灭世事件，不仅仅是毁灭地球上的人类文明，在南瞻部洲上对应着地球的那块地上的那些生命也同样在劫难逃。我们在《南瞻部洲（5）：宝塔神兵》中提到过，南瞻部洲那个地上面有一个很大的海，它就是地球在那个空间的体现，而那个海里有十九块地，地球上所有国家都在这十九块地上。我们从七印书卷看到，当地球被众神从南海那里调了差不多一滴水珠把地球淹没的时候，在南瞻部洲这十九块地的上空也下起灭世的大雨从天而降，那个雨是那里的生命从未见过的，象子弹一样直击而下，迅速就把这十九块地淹了，而那些能飞的生命如果想飞出去，如果是注定要死的，就会被空中的雷电击中。人类空间留下了一些生命，那个地方也同样留下了一些生命。

中国这个地方在上一次人类文明时期是道家文化盛行的时代，文字也是象形文字，但在上一次人类文明时期是没有纸的，是用专门的刻刀刻在专门制作的片形物品上，或者用笔写在上面。如果拿今天的人类生活方式和上一次人类文明被洪水灭世时候的生活方式相比，那时候人类生活还是很单纯的，没有今天互联网给人带来的那样混乱的思想，也没有现代这些令人眼花缭乱的智能设备和各种千奇百怪的夜生活，感觉现在的人类社会如果按那时的灭世标准来衡量早就应该被灭世好多次了。

当初在昆仑山等高山上存活下来的本次中国人祖先，他们虽然思想中有上次文明时期各种知识，但因为物品的制作是一个需要社会整体技术相互配合的产物，就象今天的芯片制造一样，很多国家、那些学这一门的老师学生都知道怎么造，但如果没有那些相互配合的工艺和产品，这个芯片他就造不出来。大洪水把上一次文明时期所有用来制造物品的机器之类的全部毁灭了，所以那些存活下来的他们也是经历了一段比较艰苦的生活，是在什么都没有的情况下从新开始的，所以虽然也能盖房子，但已经和上一次文明末期盖的房子相比不一样了，造一个火折子也不是那么容易了。然后随着一代代繁衍，因为很多关于上一次文明的那些知识也都用不上了，于是很快被后人遗忘。

但这也是必然的，因为每当新一期文明开始后，上一次文明的知识很多就被神有意的不让它们出现在新一期文明历史中，以免对新一期文明时期所要塑造的文明产生干扰。而且，无论东方还是西方，上一次人类文明幸存下来的人，他们被安排的使命其实不是来开创新一期文化历史的，他们的使命是被安排来繁衍出新文明所需要的那些人以及准备一个社会环境。这段繁衍人准备社会环境的历史大概是五百年，东西方都差不多是这个时间。这些幸存下来的人们在这五百年的历史中也有非常多的精彩故事，而且很多事情看起来也是非常了不起的，但因为他们的这些事迹不属于新文明时期需要的，于是也都不被记录下来。本次人类文明在大洪水灭世后五百年正式开始，也就是距今约四千五百年前，所以司马迁写《史记》的第一章就叫作《五帝本纪》是很对的，而后人在前面加一个《三皇本纪》就显得有些画蛇添足了。我们这里特别强调一下，中国历史所说的三皇那些人都是有的，但他们是史前某一个历史时期的历史人物。

因为青藏高原的气候变得越来越不好，这是其中原因之一，当然也还有其它原因，反正就是很多中国人就从那个地方迁移到各地。那时候还没有人称王，地大物博，你想去那里就去那里，也没人管。一起离开的一般都是有血缘关系的，或者关系好的，大家一起搬到一个新地方开始组建新的生活，所以中国人历史说氏族那也是很对的，就是很多个氏族散布到中国各地。另外，当时的社会无论哪个氏族，都很相信修炼的事情，当时大概一半的人是普通人，然后另一半的人都和修炼有关，他们或者入世学道练武学文，或者出世修道。出世修道的人到处都有，也有在上一次文明时期就修炼的高人出来世间行走。我们从七印书卷看到，那个时候的社会出来很多有功能的、开天目的人，可以说整个社会都处在这样的道家文化之下。在中国历史上有个人神共通时代的说法，于是不少人就认为这就是人神共通时代了，其实还不是，这只属于整个社会道德较高很多人都相信修炼也有很多人修炼而带来的整个社会表现出来的状态。

中国所出现的那个人神共通时代，是因为中国那个地方被打开了一条通往四大部洲的通道，然后中国那里的普通人用脚就可以走到那边，那边的生命也是用脚就可以走到人间来，是这么一个特殊的历史时期。在本次人类文明时期古希腊也出来了很多神话故事，似乎也有一个人神共通的时代，但那是无法和中国所处的这个相比的。当初在古希腊为了留下神话，只是让一些特别的人进入了另外空间一个非常有限的范围。但在中国那里却是普通人就可以去，甚至有不少的人是游遍四大部洲的，而且很多人都能够到《山海经》所说的海内经的范围之内，那边也有很多生命来到中国。如果《山海经》上面写满了那些曾经从南瞻部洲进入中国的生命物种的名字的话，那些名字装满一屋子都不止，绝非《山海经》所写的那一点。

中国本次人类文明出现的这个人神共通时代，是本次地球亿年历史从未发生过的事情，这是头一回，是为了创世主带着他的众多授记弟子从昆仑山之顶转生中国那个地方留下神传文化而特别打开的。

我们从七印书卷看到，当历史来到约四千七百年前，也就是创世主从昆仑山顶临凡之前的两百年，三界之外很高层次上有位道，他是穿白色道服拿白色拂尘的，创世主给了这位道两个指环一样的法器，每个指环有两个孔戴在两个手指上，然后这位道就来到三界最低层四大部洲的上空。只见他双手这边划那边划，于是就看到四大部洲从原来那个位置移到中间连接在一起了。我们在讲《南瞻部洲（1）：天圆地方》说过，在四海中间用来划分四海的那个纵横分界线的交汇处是魔王所在，它象巨大的火山口那样的存在，海水进不来，里面深不见底，当这四大部洲被这位道移到这里后，四大部洲的交汇处就覆盖在这个火山口上。我们知道，原来四大部洲都是被四个大海包围着的，四个大海之大，不是四大部洲里的生命随便就可以飘洋过海的，所以四大部洲的生命几乎是没有往来，除非有特殊能力，但现在四大部洲连接起来了，这对于当时生活在四大部洲的生命来说也都是一件天大的事。虽说四大部洲连一起了，但实际上真正能够用脚从这一部洲走到另一部洲的就是南瞻部洲、西牛贺州和东胜神州，北俱芦洲虽然也连在一起了，但一般生命也还是去不了。

当四大部洲连在一起后，这位道就来到中国吕梁山脉所对应的地方，然后双手一划，就打开了一条中国与四大部洲相通的通道。这条通道不是一个洞，而是整个吕梁山脉都打开了。

# 七印解封（28）：礼仪之邦的来源

2020/10/02
吕梁山脉一带在今天中国历史可能不属于什么名山大川，但在那个时候，因为人神共通从这里打开，那是远近闻名了。这个地方打开后，中国这边的人如果走到这个地方，他其实已经踏在了南瞻部洲的地上面，但他可能还是以为自己走在之前的地上，走了一会，他就会看到很多奇怪的动物，是之前从未见过的。

《山海经》里面记录的就是南瞻部洲地上面的山川河流的事情，它不是大禹写的，是人神共通之后那时候的人记录下来的，当时记录下来的很多，然后流传下来的这些东西在大禹治水的时候又从新整理了一遍。但后来很多都失传了，今天人们看到的《山海经》只是当时记录下来的一小部分，而且后人打开《山海经》方式也不对。我们知道现在《山海经》第一篇叫南山经，这就已经错了，其实正确打开的方式是“中山经”。中山经说的“薄山”是南瞻部洲上对应到“吕梁山”的那座山。因为南瞻部洲很多事情是和人间对应的，那些山也差不多，人这里有条河，它那里也有条河，比如中山经说“共水出焉，而西流注于河”，那条河就是对应到中国的那条黄河，那条河的河神就是我们在“长江黄河另外空间所见”说的那位。中山经说薄山之首曰甘枣之山，它那里有一种兽叫nuo，我们看到那个地方确实有这么一种动物，它的耳朵嘴巴和身体大小象松鼠但身体和头上、脸上的纹路都象老虎，如果把这种动物的心入药是能够治好瘿病。

因为本次文明开始的前五百年是为本次文明历史的开创准备人和社会环境的，所以到人神共通开始的时候，中国人的繁殖能力是很强的，出生的小孩一般都能活下来，于是也就是到约四千七百年前，中国已经到处都有人，组建出很多部落，北边到蒙古，东北到黑龙江、朝鲜，南边到广东福建，西边到广西云南，到处都有从新疆西藏高原那一带迁移下来的人。而这些人无论在那个地方，东北也好，还是中原地带河北河南，还是被称为南蛮之地的两广云南，人们穿的衣服和住的房子在那个时候都差不多，所用的技术也都差不多，都是从上一次人类文明留下来的文化体现。

那时候人们穿的衣服并非有些书上写着说披着树叶裹着兽皮，其实也都是很好看的衣服，但不是用今天这些化学工艺造的，他们是从一种树叶那里抽丝出来的，也有从树皮那里抽丝出来，而今天人们知道的麻布衣就是其中一种，但从树叶抽丝出来做成的衣服现在似乎看不到，估计这种技术已经失传了。另外，武士都有佩剑，也有盔甲，但盔甲不是用铁造的，而是用动物的骨头把它融化了添加一些矿物质然后按照身体需要保护部位的形状凝固成形，那个盔甲很轻也很坚固，也有象诸葛亮七擒孟获里说的藤甲。

其实今天中国广西云南那里的少数民族保留了一些上一次人类文明的文化，例如他们的服饰和房子。上一次人类文明末期中国这个地方的房子也象广西云南那些少数民族的房子，也是一盖好几层，没有中国四合院那种讲究，衣服上饰物的味道也似乎是那个样子。当然，现在广西云南看到的少数民族的房子比人神共通开始前中国人盖的房子要有规则的多，那时候可能是因为受道家修炼文化影响，我们看到他们盖的房子是没有太多讲究的，盖成圆的也有，盖成方的也有，不方不圆的也有，围墙弯弯曲曲的也有，反正就是个人喜欢，而穿衣服上也不是太讲究对称匹配，武士的盔甲有时候经常左右不对称，就是都好像比较随意，人与人之间也没有太多礼节上的讲究，例如见面打招呼，有很多人就是闭着嘴用力吹气那样。

但所有这些都在人神共通之后彻底改变了。人神共通对中国的最大影响是给中国人带来了一种从未有过的礼法。我们知道中国历史上被称为“礼仪之邦”，人们的穿衣打扮、言行举止、吃喝拉撒、房屋建造的考究，以及王侯将相士大夫的礼制、将军士兵的盔甲旗帜、行军布阵，等等，都有完整的、非常讲究的“礼法”，它们从哪里来？其实它们不是谁发明出来的，或者是经过了一个漫长的历史人们自己形成的，或者哪位神仙下凡传给人的，都不是，这个“礼法”它来自于南瞻部洲上面那个中国城。

南瞻部洲上面的生命不象人间那么迷，那里的生命知道玉皇大帝，知道下雨是龙王在上面施雨，也经常可以看到天上的神，不仅如此，本身那个地上就有麒麟、凤凰这样的神兽神鸟，也有很多天生就有特异功能的人，不少生命可以来往于天地之间。反正是和人间完全不同的一个世界了。那个“礼法”在南瞻部洲上面是必然的，就是那样的，就象天宫在地上投影一样。所以当土里的中国人到了这个地方，那是被惊呆了，美不胜收，天上宫阙，反正就是全部把南瞻部洲上有的，只要能学到的都搬到人间来，于是土里的中国人在历史上就象突然就有了一整套体系严密象天上一样的礼法，有了皇帝要戴什么帽，皇宫应该怎么盖，文臣武将应该怎么样，人与人之间应该怎么个来往，男人衣服怎么个穿法女人衣服怎么个穿法，怎么行住坐卧，等等。于是，在土里埋着的中国就出现了本次地球亿万年历史以来从未有过的“礼法”，把中国这个地方搞的象天上神仙一般。



当然，人神共通的通道能打开，也是因为土里中国人整体上处于一个道家修炼这么一种状态，有很多修炼人，也有很多出了功能神通的，而且当时中国人学的武术也是非常厉害的，就象历史书说的，有些人能擒龙灭怪，李元霸力大无穷，那时候也是有的。而且人类是一个特殊的空间，虽然土里的人出生时无法和南瞻部洲地上面的人相比，那上面的生命出生用人的语言来说几乎都是“生而神灵”，所以对于普通人来说就本事比较大，骑个麒麟也不是什么很难的事。但被埋在土里的中国人通过修炼就很快超过他们，所以土中的中国对于南瞻部洲上面的生命来说也很神奇，也有很多那边的生命来到人间，于是大家就这么相互来往。人神共通时代刚打开，无论对于南瞻部洲还是对于土里的中国人来说都无比的惊奇，但久了也就习惯了，慢慢也就觉得本来就是这样的。

另外打开通道之后那位道他就要一直在那里不能离开，他要一直负责着，做很多事情，这样两个不同空间的生命才能够相互来往。

不久的将来人间是否也会出现一个这样的人神共通时代？我们很期待！

当初吕梁山下有两个氏族，一个在吕梁山西边靠南，一个吕梁山东边靠北，西边靠南的氏族称为神龙氏，东边靠北的称为烈山氏。中国人说“近水楼台先得月”，这两个氏族最靠近人神共通的通道，而那些远近氏族要想去南瞻部洲地上面都要经过他们的地方，于是，神龙氏和烈山氏很快就成为当时整个中国最有影响力的两个氏族，其中神龙氏的统治范围一直到新疆那边整个中国的西部，包括西藏、青海，也包括蒙古，烈山氏则包括河北、河南、湖南、湖北、山东、安徽等东南部。那时候的西北不像现在那样，也不是象春秋战国秦汉所描写的那样是少数民族居住之地，那时候是当时中国最发达的地方。而后来被称为南蛮之地的两广、云南等地，可能是因为离吕梁山较远，或者是神有意而为，反正他们和人神共通的事情似乎关系不大，不在神龙氏和烈山氏范围之内。

当历史来到距今约四千五百年前，象神话一样的社会环境都已经备齐了，创世主就带着他众多的授记弟子从昆仑山之顶下世，开始了为人类的得救开创辉煌的中国神传文化的征程。

#  七印解封（29）：黄帝学艺

2020/10/04
创世主从昆仑山之顶带着众授记弟子转生中国后，创世主当时转生为黄帝。史记《五帝本纪》说“轩辕之时，神农氏世衰。诸侯相侵伐，暴虐百姓，而神农氏弗能征。”确实是这样。从人神共通到黄帝出生，历史过去了约两百年，在中国这片土地上大概形成了约136个大大小小的诸侯国，而当初最强大的神龙氏已经无法臣服它原来所瞎管的各大诸侯，而且很多诸侯本身的国土面积也都已经和神龙氏差不多，有些甚至超过了神龙氏，这些诸侯互相打，就象后来的春秋战国那样。但当初和神龙氏一样强大的烈山氏经过这两百年的发展不仅没有衰败，反而更强大了，于是烈山氏就想着征服原来属于神龙氏的那些诸侯国。黄帝就是在这么一个历史背景下出生的。

历史上关于黄帝的记录不多，也比较乱。《国语·晋语》说：“昔少典娶于有蟜氏，生黄帝、炎帝。黄帝以姬水成，炎帝以姜水成“。我们没看到黄帝和炎帝是两兄弟，我们只看到炎帝属于吕梁山以东的烈山氏，黄帝属于吕梁山以西的神龙氏。史记《五帝本纪》说：“黄帝者，少典之子，姓公孙，名曰轩辕。生而神灵，弱而能言，幼而徇齐，长而敦敏，成而聪明。“我们看到，黄帝出生在陈峰氏这么一个部族中，当初这个部族属于神龙氏的一个分支，有国号，应该相当于诸侯，面积不大，后人称之为少典，但我们从七印书卷看到这个字很象沙字，也就是“少”字前面有一个三点水，而典字前面也有个三点水，但为什么后人把三点水去掉我们不是很清楚，我们就按照现在历史的说法叫少典。《五帝本纪》说“少典之子”也没错，因为黄帝的父亲是这个少典的国君。少典国的位置大概在今天陕西省延安市和榆林市之间的某个地方。

当黄帝长到五岁的时候，他的父亲少典国君就张榜设下三大关为黄帝招聘启蒙老师。第一关要打败少典国最强大的武士，第二关要破解少典国最厉害的阵法，第三关要博闻强记。来应聘的人很多，但没有一个能够通过这三关的。这一天，黄帝家来了一位应聘的，他长得有些象三国演义图书画的那个老了的邓芝，是在横断山脉里修炼的一位道人，他是专程为黄帝而下山的。第一关，那位道人来到比武场上对着少典国最强大的武士拱手说了一声“请”，然后那个武士就使出浑身武艺攻向道人，但大家都没看到道人怎么出手，反正就是武士刚要使出招数就被击倒了。然后第二关，少典国将军拿出一个象围棋盘那样的器具，里面摆了一个他们认为最厉害的阵法，这位道士看着阵法就大笑说这也太简单了，然后举手间就把这个阵破了，跟着这位道士在上面也摆了一个阵法让少典国将军看，少典国将军只看得奥妙无穷无法破解。第三关，少典国见识最广的一位智者拿出一本古老的书要这位道人先读一读，然后他会根据这部书的内容提问题，但道人却说这部书不用读了，你就直接问吧。结果无论那位少典国智者问什么问题，这位道士都能回答。于是，黄帝父亲设下隆重的拜师礼，让黄帝拜这位道为师。

这一学就学了七年，黄帝十二岁的时候，有一天这位道来见黄帝父亲辞行，说自己这七年只是教了黄帝一些基本功，黄帝还需继续拜师学艺，然后交给少典国君一封信，让黄帝带着这封信去某个地方找一位高人学艺。这个地方在今天看大概在山西运城一带。少典国君就按那位道人的吩咐，给黄帝准备了一匹好马，带齐各项物品，择了一个吉日就让黄帝只身一人远赴他乡学艺去了。

黄帝马不停蹄地按照道人所说来到了一座城，城里有很多人，很繁华，他也不知道自己要找的那位高人在那里，城里的人都不知道有这么一个高人。黄帝就一路问来到一个铸剑的地方，那里挂着很多宝剑售卖，里面有一个白胡子老人，这些剑都是他打造的。黄帝就问这位白胡子老人，问他是否知道某某高人，那位老人就说“知道”，黄帝大喜过望，就问是否可以告诉他，老人说“可以，他在”，然后老人把脸朝向左边，黄帝立即着急地用手指向左边说"哦，是在那边"，老人说“不是”，黄帝又立即着急地用手指着右边说“在那边”，老人笑着对黄帝说“你不要着急，你要静下心来听我说”，于是黄帝只好按捺住性子，然后老人说，“你要找的人就站在你面前”。黄帝大喜过望，连忙把第一位师父的信交给这位老人，老人看完后就收了黄帝做徒弟，教授黄帝驾驭水火的道术。于是黄帝就跟着白胡子老人一边铸剑卖剑一边学习道术。

这一学就学了四年，黄帝十六岁的时候，已经学会了驾驭水火的道术，水不能溺火不能焚，也能踏水而行，也能口中吐火口中吐水。这一天白胡子老人对黄帝说你已经可以出师了，但你还需要到某个地方找一位高人跟他继续学艺，然后就给黄帝一封信和一把神剑。这把神剑可以变化，不是用人间的铁打就的。《中山经》里描述了南瞻部洲上面有一座山叫鲜山，山中有一种动物叫鸣蛇，状如蛇而四翼；这把神剑的剑身就是用这种鸣蛇的皮做的，而剑柄则是南瞻部洲上另一种动物身体里的一种物质提炼出来的。于是黄帝就带着信和神剑去寻找他的第三位师父了。

他的第三位师父大概是住在今天河北石家庄那一带，那里当时也是一座城市，人口很多。黄帝来到这里后找了很多天都没找到这位师父，那里的人都不知道有这么一位高人。这一天黄帝看到前面有个肉铺，于是他过去问卖肉的是否认识这么一位高人，那卖肉的说知道，黄帝很高兴，就打躬作揖请问高人住在何处，卖肉的就详细地对他说你往左走，然后看到什么就右转，然后看到什么就左转，反正告诉了一大段话。黄帝听完拔腿就跑，按照卖肉的所指的路线一路走来，结果发现自己又走回到原地看到肉铺，很奇怪，就问难道走错了我怎么又回来这里了？卖肉的就笑着说，你面前的就是你要找的那个人啊。于是黄帝就把第二位师父的信交给这位卖肉的高人，然后这位高人就收了黄帝做徒弟，于是黄帝就一边卖肉一边学习各种拳脚刀枪剑戟等真功夫。这些真功夫可不是当时社会上将士所用的，看起来象是天兵天将所使用的，例如其中一种剑术，黄帝双手各执一把剑，人就飞起来了。

这一学就学了五年，黄帝二十一岁的时候，这一天第三位师父告诉黄帝说你可以出师回家了，去做你该做的事情。于是黄帝就带着满身本事辞别师父回到少典家中，然后不久少典国君就对他说我们现在神龙氏面临很多问题，内部诸侯互相攻击，外面烈山氏虎视眈眈，你既然学了满身本事，那就去求见神龙国君，出一份力。于是黄帝就离开了少典国，来到了神龙氏都城所在。

# 七印解封（30）：黄帝获麟

2020/10/07
黄帝为了历炼自己，他离开少典国去神龙氏都城是走路的，没有骑马。神龙氏都城一开始在吕梁山之西靠南，后来在黄帝出生前搬了一次都，搬到了今天内蒙古鄂尔多斯一带，那时候那一带地貌和现在不同，那些地方都是肥沃之地，到处都是森林、河流湖泊，人口众多。我们估计神龙氏搬迁也可能和烈山氏势力越来越强盛有关，因为这时候烈山氏已经占据了河北、河南，山东等地，今天陕西运城那一带也属于烈山氏，离神龙氏原来都城很近。

这一天黄帝来到一个客栈住下，他晚上打坐时体察出附近有异常，于是他右手握神剑，凝神定气。屋里突然出现一个黑影在他周围快速跳动，速度很快，然后只见黄帝神剑出手，跳动那个身影的脖子就被神剑象一把绳索一样套住，黄帝一看原来是一只猴子，他立即左手一掌把猴子击飞，然后右手一剑刺向身后，只听啊一声惨叫，从黑暗中一个人中剑倒地，跟着出来一队武士攻击黄帝，但敌不过黄帝武艺高强，这些武士就败退逃走了。被黄帝神剑击中的倒地之人并没有死，于是黄帝就为他疗伤，治好了他。这人很感激，黄帝问他叫什么名字，他说自己没有名字，于是黄帝就赐了一个名给他，为了行文方便我们这里把他叫“黑者”。

那时候中国和现在很不一样，有很多各门各派的练武之人，都是真功夫，不是现在那些花拳绣腿。在一个原始森林里有一个练武之人，他所练的很象今天知道的日本忍术，我们估计日本忍术可能就来源于他那里，具体怎么个来源我们就不细查了，因为查一个事情的来龙去脉真是很麻烦的一件事情，我们就把他这一门称为忍术算了。他带了一帮徒弟，都是无名无姓，象忍者一样生活，他们所学的很多都象日本文化里说到的各种忍术。黄帝身上他第二位师父送给他的那把神剑可以说是世间至宝，这个忍者他有一天看到了黄帝身上的那把神剑，他就动了邪念想得到它，这些偷袭黄帝的就是他派来的。我们也看了一下，黑者右肩膀上有创世主的授记，他其实是追随创世主从昆仑之顶下来的。这个黑者在历史书上没有提及，因为他跟随了黄帝后就作为黄帝的贴身护卫了。

黄帝带着黑者离开客栈就一直往北走，这一天来到一个镇上，有很多人，当他们走过一个肉铺的时候，听到卖肉的喊道，“卖肉啊，有猪肉、羊肉、牛肉，还有麒麟肉、寓鸟肉”。黄帝听到了“麒麟”，黑者听到了“寓鸟”，两人都大吃一惊。麒麟和寓鸟都是世间罕见之物，怎么在这里被人做成肉叫卖？因为黄帝第三位师父就是个卖肉的，所以黄帝心想或许这位卖肉的也是高人，于是就请教卖肉的，那位卖肉的笑着说我这里没有麒麟肉，也没有寓鸟肉，但如果你们想要，我知道那里有。黄帝听师父说过将来如果有机缘一定要以麒麟为骥，而黑者则听他的师父说过如果忍者能得到一只寓鸟那是忍者最好的护甲了，于是黄帝和黑者几乎异口同声请求这位卖肉的带他们去。卖肉的也很爽快，马上就把肉铺关了，带着黄帝他们七弯八拐的来到一山中。其实这个卖肉的就是一位高人，他是专门在这里等黄帝的，喊卖麒麟肉、寓鸟肉也是因为看到了黄帝就故意引他们。这位高人对黄帝说这山里就有麒麟，问黄帝知道怎么驯服麒麟之法吗？黄帝说不知道，然后这位高人就教黄帝怎么驯服麒麟之法。于是黄帝和黑者就走进了山中。具体山中发生了什么事情我们就不细说了，因为这样细说下去我们就不知道什么时候才能写到那个终局了。反正过了一段时间，黄帝就骑着一只紫色的麒麟从山里走了出来，而黑者肩膀上也站着一只寓鸟。

麒麟按能力来分大概可以分为三类，一类叫地麒麟只能在地上跑不能飞，第二类可以称为飞麒麟因为它可以飞，第三类可以称为天麒麟因为它可以变。黄帝得到的这只麒麟就属于天麒麟，南瞻部洲地上面都是罕有的，更不用说埋在土里的中国了。每种麒麟怎么收服它也是很有讲究的，都有一套收服的方法，黄帝从那位高人那里知道了怎么收服麒麟之法，而怎么收服寓鸟，黑者的师父也教过他，因为这是忍者梦寐以求的。《山海经》北山经说有座虢山，山上很多寓鸟，状如鼠而鸟翼，其音如羊，可以御兵，说的就是它。《山海经》书上说能御兵但怎么御兵没有说，其实这种鸟身上能够发出一个能量罩，把那些攻击的兵器挡在外面。

那位高人很高兴， 就问黄帝你要去哪里？黄帝说要到神龙氏都城，高人就说从这里到都城有五条路，每一条路上都有拦路的，不好过，你们想走哪条？黄帝问怎么个拦路法？高人就说第一条路上有一个火人，第二条路有一个怪兽，第三条路有一个功夫很高的人，第四条那里有一个很大的悬崖要飞过去，第五条路有一位道人他会出五道难题。黑者马上说走火人那条路，因为黑者懂忍术之中的火术，他的嘴能吐出火，而黄帝也同意，因为黄帝第二位师父教了他御水火之术。于是那个高人用手一指，那你们就往这个方向走。

可是，当黄帝他们按照高人所指的走着走着，他们发现前面路中间坐着的是位出五道难题的道而不是火人。黄帝知道这一关自己是必须过了，骑麒麟飞也肯定是飞不出这个道的手掌心，只好前去向这位道打躬作揖，问这位道自己要从这里过。那位道说你要想从这里过就把你身上那把宝剑给我，或者你们能回答我五个问题，如果第一个问题回答不出来你们就要被鞭打一百鞭然后回答第二个问题，如果第二个问题回答不出我就要从你们身上剐去一块肉然后回答第三个问题，如果第三个问题回答不出来就要把你们放进油锅炸半个时辰，如果你们能活下来就回答第四个问题，如果第四个问题回答不出来你们就要被火烧半个时辰，如果你们能活过来就回答第五个问题，如果第五个问题回答不出来，我就要取走你的宝剑。黄帝问怎么个五道难题，那位道就用手一指远方说你看那边那棵树叫什么名字？黄帝一看那颗树不是人间有，但好在他第一位师父博闻强记，关于这种树的知识也都教过，于是就笑着说这棵树叫什么什么。于是那位道接着出第二道问题，指着远处的森林说你们要在一招之内把那边五十棵树都砍下来，黑者说这个我来，然后从身上取出一把象飞镖那样的武器一下就把五十棵树砍倒了。于是那位道说你们要给我表演一套人间没有的武艺，黑者不知道这是什么武艺，但黄帝知道，因为他第三位师父教的都不是人间武将所用的武艺。他叫黑者把他背上的两把剑给他，于是黑者从背后取出两把剑交给黄帝，黄帝手拿双剑武了一套天兵剑法。于是这位道就把拂尘往空中一抛说，你们在一炷香的时间内要抓取这个拂尘，黑者一见就立即说抓东西是我的看家本领，然后上前一抖身很多暗器之类的就从身上飞出来击打拂尘，但拂尘好像长了眼居然避开了所有这些暗器，于是黑者施展他的追踪术追击拂尘，但无论怎样也追不上，眼看时间要到了，黄帝叫黑者退下，然后从腰上取出神剑抛向拂尘，神剑变成爪一样飞出去就把拂尘抓住了。于是那位道就说最后一个问题是你们要打败我。黄帝只好和黑者一起围攻这位道，但怎么也打不败他，黄帝只能认输。那位道就说那你身上的神剑就只能归我了，黄帝虽然百般不愿意，但也没办法了，只好解下宝剑递给这位拦路的道，然后这位道就放他们离开了。

我们把这个事情的因缘看了一下，原来这位道就是那位卖肉的高人，他是一位修道之人，当他修到一定层次将要往更高层次突破的时候，他看到一位神，这位修道人在某个历史时期曾经欠过属于这位神的债，这位神要这位道还了这些债才会放他提高到上一个层次，还债的方式就是这位道要生活在这个镇里收集一定数量的宝物给他。当黄帝路过这个地方时，他那个债主就说黄帝身上那把宝剑抵消后面所有的债，于是这位修道人就先让黄帝得到麒麟，这样他即使拿走了黄帝的神剑他也不算欠黄帝的， 否则旧债才了又添新债，然后再让黄帝他们去过自己设下的五关以取走黄帝的神剑。

黄帝他们往前走了一段路，就看到自己第二位师父，也就是那位打铁造剑的白胡子师父，他手中拿着那把已经给了拦路道人的神剑站在前面等他，黄帝连忙带着黑者上前施礼拜见，师父把神剑交还黄帝说在取出神剑交给那位道的时候我使了一些法术用一把假的把真的换回来了，但对于他们来说那把假的也是真的一样。

黄帝辞别了师父后就带着黑者骑着麒麟飞起来，可是飞了很长时间却好像还在山里，他们正觉得奇怪的时候，只见这只麒麟就慢慢下降，当下到地上的时候它就变成了一匹马。这种天麒麟它是有灵性的，它自己会根据不同的空间不同环境变化不同形象。这匹马停下来也不往前走，就是不断地对着前面喷气，于是黑者上前从腰中取出一个爆炸物往前面空间扔过去，只见那个爆炸物一下炸开但之后又慢慢恢复原样。黑者说前面不能走，有高人设下了一个空间。于是黑者不断用他的爆炸物在前面探路，黄帝就骑着马在后面跟着，七弯八拐的就从里面走了出来，回头一看，那座山没有了，他们是在一条通往神龙氏都城的大道上。

原来黄帝他们去到的那座山其实是在南瞻部洲地上面。

#  七印解封（31）：龙帝

2020/10/10
神龙氏都城按今天来看，可能相当于现在中国一个县城那么大，有城墙，高约两层楼，这在当时已经属于很大的城了。当时中国有两个皇帝，一个是烈山氏称帝，称为炎帝，一个是神龙氏称帝，称为龙帝。他们都头戴皇帝的冠冕，前面有冕旒，而其它所有部落氏族都是王公侯等诸侯部落，而那些比较强大或者与神龙氏或者烈山氏有比较久远渊源的诸侯就戴王冠称王，黄帝的父亲少典属于侯。有些书上说与黄帝同一时代的炎帝长着牛头人身，头上两个牛角，我们没看到这个样子的炎帝。当烈山氏炎帝传到创世主临凡转生为黄帝这一代的时候，炎帝的样貌也是人样，和三国演义图书里的那个董卓长的很象，也那么个高度，也那么胖，心也比较狠，不是善类，他手下有很多象吕布那样勇冠三军的虎狼之将，所以在当时来说，炎帝在武力上是很强大的。而神龙氏龙帝传到这一代的时候，则属于软弱之辈，没有主见，怕事，贪图一时的安乐，所以对于炎帝吞并中国的野心，龙帝自然就是一退再退。

黄帝来到神龙氏都城后，先找了一个客店住下，店名“小王都”。第二天一早就来到龙帝皇宫外求见，但不巧这一天龙帝酒喝多了不能接见，于是黄帝只好回去再等。又过了几天，黄帝又来到皇宫外求见，这时候龙帝还在睡觉没法接见，黄帝只好又回客店。又过了几天，黄帝又来见龙帝，这时候龙帝正在宫中和别人喝茶聊天不想见，黄帝又只好回去了。又过了几天，黄帝再次来到皇宫外求见，这次龙帝恰好什么事都没有，于是就召见了黄帝。龙帝问黄帝三番四次前来求见所为何事，黄帝就介绍了自己所学本事，又说了一些炎帝想要侵凌诸侯的野心，又说自己想来当一名将军帮助龙帝攻打炎帝，龙帝一听要和炎帝打就连邹眉头，就说自己这里不需要什么将军，你回去吧。黄帝非常伤心，但没办法只得离开皇宫，一路上走，后面皇宫守卫就不断催促黄帝赶紧走。黄帝正在伤心之时无处发泄，于是那些守卫越是催，黄帝就越是慢慢悠悠地走，有一个士兵气不过，不知道手中拿了一个什么东西就扔向黄帝，黄帝也不回头，没看见一般继续往前走，等那个东西差不多到自己头的时候，他就象背后长眼睛一样头突然一则就避开了，然后大笑着快步离开皇宫。

几天后，黄帝就带着黑者离开了神龙氏都城。都城门外有一个分岔道，一条往西，一条往南，因为没达成父亲的愿望，也就不想回少典了，黄帝就骑着那匹麒麟变成的白马带着黑者往西方那条道去。每一件事情背后如果往深去看，会发现真是奥妙无穷，其运作方式和我们看到的有时候完全不一样。麒麟什么时候变马什么时候显现为麒麟不是由黄帝决定的，好像是由麒麟自己决定的，但我们又细看了一下，发现其实也不是由麒麟决定的，是天上有一位神他好像就是专门来负责这类事情的，当他认为麒麟需要显现为马时，这位神就会说“变成马”，然后那只天麒麟就会变成马。而黑者那只状如鼠鸟翅的寓鸟不会变，它原来是什么样现在还是什么样。

黄帝他们顺着道大概走到今天内蒙巴音乌索镇一带的时候，听到前面树林有打斗声，于是催动白马来到树林里一看，原来大概有几十个强盗在围攻十多位身穿盔甲手拿盾牌的将士，情势危急。那些士兵都是步行，手中的盾牌有一人高，不是用铁做的，不知道是用什么做的，半透明很轻很坚硬，他们用盾牌组成一个防御阵。领兵的将军穿白色盔甲骑着白马，背上有一把用布紧紧包住的宝剑，手中拿着一种武器，我们不知道那是什么武器很少见，柄很长，头部那里象锤，锤底部又有两柄剑一样横在那里，锤顶部又有一把剑伸出来那样。强盗头子身上没有盔甲但脚下踩着两条蛇手拿一把大刀，他的脚就象和蛇连在一起稳稳地踩着速度很快。他带着六个步行副将围攻骑白马的这位将军，将军武艺高强，但在七位高手围攻之下也终是不敌，眼看这位将军就要被强盗头子一刀砍中的时候，黄帝见情势危急连忙飞出手中神剑，只见神剑化为闪电一般一下就把强盗头子的大刀隔开，然后黄帝和黑者冲过去帮将士们打退了强盗。

将士们非常感激黄帝的救命之恩，大家也互通了姓名。那位将军姓白，我们就叫他白氏。白氏说自己是汉氏部族的，护送背上那把宝剑到汉氏都城，路上不知道怎么被这些贼人看中了，于是他们在这里拦路想抢夺宝剑，然后白氏问黄帝从哪里来要去哪里。黄帝说自己来自少典，去神龙氏都城想为龙帝出力对抗炎帝，但不被龙帝赏识，现在就随便走走没想去哪。白氏很高兴，说您武艺高强，我们汉氏部族虽然不是很大只有两座城，但我们主公礼贤下士，汉氏都城离这里不远，你不如和我同去见我们主公，我给你们引见，如果你喜欢就留下，不喜欢你再走也不晚。黄帝就说好啊，于是白氏就带着黄帝去到了汉氏都城。

汉氏都城大概在今天内蒙赛因乌素苏木靠近黄河那一带，就象今天一个地方小镇。白氏他们进城后直接带着黄帝他们去见汉氏君复命，并把路上遇到强盗拦截被少典之子救下之事说了。汉氏君穿白色侯服，见黄帝气宇轩昂非一般人物，很是喜欢，就问黄帝有什么志向。黄帝说了自己想重振神龙氏以抗炎帝侵凌，但可惜不被龙帝认可。汉氏君就说真是可惜啊，然后问黄帝有什么打算。黄帝说自己现在也没什么打算，就是想到处走走看。汉氏君说炎帝势力越来越大，我们神龙氏各部族都非常忧心，知道迟早有一天会被炎帝攻打，而龙帝又没有作为，所以现在大家都各自为政，都在想办法自保。然后两人就你来我往地说了很多话说了很长时间，交换了很多意见，最后汉氏君好像被黄帝说服了，就说，我们汉氏就这么小的两个城，不可能实现你的志向，但我们这一带象我们汉氏这样的小部族很多，大家现在就象一盘散沙，谁也不服谁，如果你有本事说服大家联合起来，那这个力量就不会比龙帝的弱，当你有了这个力量的时候，我就带着我们汉氏归附你。黄帝就说好啊，那我就去说服他们。汉氏君说谁和你去？黄帝用手一指黑者说我们俩。汉氏君不太敢相信，就说凭你们两个就想去说服他们？这时候白氏站出来说我也想和他们一起去，汉氏君犹豫了一会就说行，那你就带着你那些将士和他们一起去试试，如果真能成功，那我们就有希望了。

于是，黄帝就带着黑者、白氏以及白氏挑选出来的二十四位精兵，这二十四位精兵全部身穿金色盔甲骑白马，马上挂着那个半透明的盾牌，一行二十七人二十七匹白马离开汉氏都城，开始了说服神龙氏散沙一般的各部族首领之旅。我们看了一下，白氏和那二十四名精兵右肩膀上都有创世主的授记，他们都是当初追随创世主从昆仑之顶下来的。

# 七印解封（32）：黄帝云师的来源

2020/10/12
黃帝在离开汉氏都城的时候，汉氏君给了黄帝一张地图，上面列明了神龙氏诸国的所在，黄帝就对汉氏君说届时那些被他说服的各诸侯将会陆续派兵来这里会盟，希望汉氏君在都城外准备一个开阔的地方，以迎诸侯，汉氏君欣然答应。然后黄帝就跨上他那匹由麒麟变成的白马，这匹白马头部那块鬃毛是红色的，尾巴也是红色的，其它地方全是雪白没有杂色。黄帝头上没有头盔或者帽子之类的，就象古人那样把头发卷起来用一个发簪，身穿蓝色的紧身衣，紧身衣肩膀和手臂那里是银色的盔甲，胸部那里也是银色的盔甲，这套紧身衣盔甲是他第三位师父送的。盔甲的外面，黄帝身上披着一件黄色战袍，是他的第一位师父送的。战袍上绣着很多云，云都是朝霞红，云的边绣着金线，一眼看去，这件战袍在当时整个中国也是独一无二的，非常夺目，于是就被当时世人看成是黄帝最明显标志。史记《五帝本纪》说黄帝“官名皆以云命，为云师”，后人不明白为什么黄帝以云命，又说是因为黄帝受天命时天上出现了云瑞的征兆，其实这个“云师”就来自于黄帝第一位师父赠的这件战袍，这件战袍在那个时候是“黄帝来了”的一个非常明显的标记，命名云师以表对授业恩师的传承。

可以想象的到，黄帝这一身打扮，带着白氏和黑者，其中白氏一身白色盔甲，黑者一身黑色盔甲，二十四名精兵全部金色盔甲，然后二十七匹白色战马，如果有会画画的把这个画面画出来，那一定是非常好看的！

当时神龙氏的范围很广阔，东北方向一直到俄罗斯哈巴罗夫斯克边疆区那里也有一个诸侯，黄帝他们离开汉氏都城后是直奔这个地方而来，他们坐下那些战马就象关羽的赤兔马一样跑的飞快，日夜兼程，一路上并没遇到什么大的阻碍就顺利到达了，但当他们来到这个地方时，黄帝看到城里的百姓被随意鞭打和奴役，于是他也不去见这个诸侯的王了，带着黑者白氏他们直接掉转马头就顺着海边往回走，然后来到了今天吉林珲春那一带，那里有一个诸侯，被称为骸氏。

骸氏也是上一次大洪水灭世后从昆仑山那一带活下来的人迁移到这里，他们的衣服房屋等等到黄帝来这里的时候也是基本保持着那个时候的样子，士兵的武器和着装也基本是那个样，所以和黄帝他们的装备比起来就显得比较粗糙。他们的武艺也很高强，普遍力量都比较大，所以武器上多是大锤子、大斧头、大刀之类的。可能是距离遥远的原因，人神共通的礼仪文化并没有影响到这里，所以当城里的人看到黄帝他们的时候，都惊为神仙下凡，纷纷上来围观。另外，这里百姓走路的样子很奇怪，就象人一瘸一拐那样走，而那些能正常走路的，一般都被训练做为将士了。我们还发现这里约10%的人嘴唇中间有一个很小的骨头突出来，样子有些古怪。我们查看了一下来源，发现原来在几百年前从烈山氏那里来了一个厉害法师，他把这里的人当实验品，试练一种法术，让这里人长的象骷髅一样，后来这里的人就以为自己是骷髅人，所以称骸氏。随着时间推移，那个法力也消失了，于是这些人就慢慢的在恢复原貌，但到黄帝来的时候仍然还留下不少后遗症，于是有些人走路就象骷髅那样，嘴唇那里有人也长出骨头。因为这个原因，这里的人对烈山氏并没有好感。黄帝见到的骸氏君样子很胖，头上戴着一个用动物骨头做成的帽子，他嘴唇那里没有突出的骨头，可以说已经完全恢复了原貌。黄帝与骸氏君相见后就说明此行目地，骸氏君很高兴，也没犹豫就答应了，于是黄帝就交给骸氏君一张地图，让他在某年某月之前带着骸氏将士到地图上标明的汉氏都城会盟，然后在那里他会给他们配备新的武器装备，并从新训练他们。骸氏君和众将士都很高兴，巴不得现在就到汉氏都城那里。

离开骸氏，黄帝他们来到了今天吉林永吉县到北大湖镇一带，这里有一座城，城门口那里写着“江工都”三个大字，被称为江工氏。从这里人的衣着居住、礼仪举止来看，虽然和黄帝他们还有一些差距，但大体上已经有了黄帝他们那样的装束和言行举止，也就是说人神共通的礼法已经传到了这里。江工氏崇尚银色，这里的将士都是统一的银色盔甲。黄帝以少典之子拜访江工氏君，江工氏君很高兴，就请黄帝前来和他的一众文官武将见面。双方见完礼后，江工氏就问黄帝所为何来，黄帝就说明自己想联合神龙氏诸侯一起征讨炎帝的来意，江工氏一听就觉得黄帝太狂妄自大了，很不高兴，指着前面一位武将说你去把这个狂妄之徒给我赶出去。那位武将一听正要从腰间拔出佩剑，只见黄帝右手一动，神剑就立即变成一条长鞭把武将的佩剑卷到黄帝左手上，然后笑着说有没有更高强的将士要出来和我比试？江工君一看大吃一惊，不敢小看黄帝了，就说你如果能够打败我们第一武士，那我愿意出兵与你会盟。黄帝说好，于是江工氏第一武士提着一把宝剑就前来和黄帝比试，黄帝也不用武器，就赤手空拳击败了江工氏第一武士，令江工氏君上下无不心服口服，于是黄帝也一样交代了江工氏君于某年某月之前带兵到汉氏都城会盟。

从江工氏往今天中国的辽宁方向，在抚顺一带有一个诸侯，他们都城的城门口写着“日都”两个字，被称为日氏。这个日氏从他们将士的穿着打扮以及这个氏族的称号来看，应该和今天日本这个民族的渊源很深。他们将士的盔甲都是白色，但胸部那里的护甲却全部带一个红色的圆，和今天日本的国旗很象。从他们的言行举止和衣着举止来看，人神共通的礼法在这里已经有比较深的影响了。黄帝给日氏君说明来意后，日氏君就说出兵可以，但首先你必须要为我们解决一个后顾之忧。黄帝问什么个后顾之忧？日氏君说我们这里最近来了一伙强盗占山为王，他们那个头领手里有一柄开山神斧，带火，什么武器都经不住他斧头的一砍。他们经常来骚扰杀害我们这里的百姓，我们多次围攻他，最后都因为他那把斧头而不能成功，再这么下去，我们日都估计都要被他占领了。黄帝说那好，我们去会会他。于是日氏君就让人带着黄帝他们去到强盗占据的山脚下。黄帝就和黑者白氏商量剿灭强盗的办法，黑者说既然他那把斧头这么厉害，就由我先上山把他斧头偷了，然后我把山门打开，白氏就带将士们攻上来。黄帝和白氏都觉得可以，于是黑者就在晚上使出他忍者的看家本领把强盗头子的斧头给偷了，然后打开山门，黄帝和白氏带着二十四位精兵拿着差不多一人高的盾牌就攻了进去，拿住了强盗头子后把他交到了日氏君手里。日氏君很高兴，就答应将在约定的时间带兵到汉氏都城会盟。

# 七印解封（33）：王亥驯龙

2020/10/14
在今天内蒙古赤峰和通辽之间有一个诸侯，它的城门口上有“调雨”两个字，被称为调雨氏，黄帝他们离开了日氏都城后就日夜兼程来到了这里。

特别说明一下，我们从七印书卷看到的文字不是我们现在看到的那样，有些类似于小篆，当这个名字是可以流传于世的，那么我们就可以看到这个字的外形，然后如果我们想把这个名字写下来但我们又不知道它是什么，七印书卷就会出来声音把这个字念出来，然后我们根据它的外形和声音查找字典比较相似的字，例如“调雨”。又例如白氏将军，他其实也是有名的，但他的名我们就看不清，属于不能流传于世的，所以我们就称他为白氏。看名字是非常费劲的一件事情，所以能不看我们就不看。

从调雨氏人的言行举止可以看到人神共通的礼法在这里已经是非常深入了，而且这里的武士武艺特别高强，一般都能以一敌十，但这个调雨氏君也是自视较高，无论黄帝怎么说他就是不愿意到汉氏都城与黄帝会盟。但黄帝很想得到他们将士们的加入，于是就带着黑者和白氏在城里转悠，这一天肚子也有些饿了，看到前面一个酒楼，好像很多人光顾，于是他们就进去叫了些酒菜坐下边吃边聊。黄帝一身带红云的黄袍非常显眼，一下子就引起了这家酒楼老板的注意，他长的就像三国演义那时候的书生，就过来问黄帝是那里人来这里做什么。黄帝就说自己是少典之子，正在周游列国想说服各诸侯和他一起讨伐炎帝，可惜无论自己怎么说调雨氏君就是不愿意，现在正想办法看怎么才能够打动调雨氏君。店老板一听就说我知道怎么打动他，黄帝连忙问什么办法？店老板说你只要给他找来一颗龙蛋他肯定会答应你。黑者一听就吵吵嚷嚷说“不会吧，怎么会有人想要龙蛋？”白氏也吵吵嚷嚷说“我们要去哪给他找龙蛋啊？”黄帝朝他们两个摆摆手说别吵，既然店家这么说了他肯定知道怎么做，我们还是听听他怎么说。于是黄帝就问店老板怎么知道调雨氏君真的想要一个龙蛋？店老板就说我这家店的饭菜在我们这里也是远近闻名了，所以有时候调雨氏君也来光顾，有一次他来我这里吃饭，我偶尔听到他和他的几个贴身侍卫说想要一个龙蛋，但他又不想告诉别人，免得大家都想着怎么给他找个龙蛋，扰了大家的生活。黄帝就问那您知道哪里有龙蛋？店老板就说我们调雨氏附近有一座龙山，上面有龙，但那些上到这座山的都没有回来的。黑者和白氏就说这有什么可怕的，我们去会会它。黄帝摆摆手又问店老板说您是否可以帮我们去山上拿一个龙蛋回来？店老板说可以，黄帝非常感谢，双方互通了姓名，原来这位店老板叫王亥。

收拾妥当，黄帝他们就和王亥骑马动身前往龙山，王亥看到黄帝的白马就说，你这匹马是麒麟吧，黄帝心里很吃惊，知道这个店老板王亥是位奇人，心想等在这里事办完无论如何都要邀王亥跟自己一起走。龙山很高很陡，王亥就带着他们绕着这座山走一些偏辟的路，有时候没路，弯弯曲曲地往山上走，然后前面有一片原始森林，他们就穿过这片森林，远远的地方依稀可以看到一个龙洞，王亥就叫大家停下来不要出声，说前面山洞里就有一条龙，等下如果大家看到那条龙闭上眼睛的时候就派一个人惦着脚进去抱一个龙蛋出来，千万别弄出很大的声音，另外记住只能抱一个龙蛋，里面无论看到什么样的宝贝都绝对不能动，否则就会没命，那些上到这里回不去的都是因为动了要拿这里宝贝的念头而没命的。于是黄帝就交代黑者到时候负责进洞抱龙蛋，不准动里面的任何东西。

吩咐完，王亥就拿出一个摇铃那样的东西从树林里走出来，一边摇着玲一边手舞足蹈地跳起舞来，那个舞很特别。也真是奇怪，王亥这边一摇铃一跳舞，洞里就出来一条龙，中国传说的那种龙，也跟着王亥跳舞。王亥这种舞实际上它是一种咒诀之类的，不同的是这种咒诀是跳出来的。王亥怎么扭，这条龙也怎么扭动身子，王亥手脚怎么动，它也手脚怎么动，王亥停在那里不动，它也停在那里不动，王亥把眼睛一闭，然后这条龙就象睡着了一样把龙眼闭上站在那里一动不动。黄帝一看龙闭眼了，赶紧叫黑者偷偷进洞抱一个龙蛋出来。黑者进到洞里，只见里面各种金银财宝各种宝剑武器之类，反正是让人看着垂涎欲滴。洞里面有三个龙蛋，黑者就抱了一个龙蛋出来。我们看到这个龙蛋很大，大概有黑者腰那么粗半个身子那么大，也很沉，黑者几乎用尽力量才勉强把它抱出来。当黑者抱了龙蛋出来后，王亥就和黄帝他们走出森林，那条龙就突然睡醒了一样，回去看只剩两个龙蛋，但它却又很奇怪以为自己就是只有两个龙蛋。

我们看了一下，调雨氏在历史上曾经出过一位高人，他懂得一种道术可以呼风唤雨，但需要龙的配合，就是因为这个缘故他们这个氏族被称为调雨氏，后来不知怎么这种道术就失传了，到黄帝这个时代的调雨氏君知道了这种道术，可惜缺一个龙蛋。现在调雨氏君得到龙蛋很高兴，就答应了黄帝的要求，在约定的时间到汉氏都城与黄帝会盟。

黄帝道别了调雨氏君后就去找王亥，说希望他能跟自己一起走，王亥立即答应。王亥又说他有一位朋友，是个樵夫，就一个人，叫贾齐，使一把斧头武艺很高强，他肯定也愿意跟我们一起走。黄帝说太好了，于是王亥就带着黄帝和黑者白氏他们去到一座山的山脚下找到贾齐住的地方。贾齐长的有些象程咬金，五大三粗的。王亥给大家引见后就说明来意，贾齐很高兴，就说你们等一会，我收拾一下马上跟你们走。不一会贾齐就背着个包袱，一身樵夫打扮，手中拎个砍柴的大板斧就出来了。黑者一看就说我这里有一把更好的，然后就从腰里一摸，不知怎么就出来一把斧头，就是在日氏那里从那个强盗手中偷来的那把能出火的神斧。我们看到黑者他们学的忍术之中，好像有一个神奇的袋子，里面很能装东西，各种弹药、暗器之类的，应有尽有，象用不完一样。贾齐很高兴，拿着神斧就跟着黄帝他们回到王亥的饭店。王亥把饭店事务全部交付给下人，交代妥当之后，王亥、贾齐就跟着黄帝他们上路了。我们看了一下，王亥、贾齐右肩膀上都有创世主的授记，他们都是当初追随创世主从昆仑之顶下来的。

现在有些故事说那时候人们还是蒙昧状态，连马是什么东西都不知道，说有一种动物叫“火畜”，王亥把它驯服就成为马，于是我们人类也就有马骑了。我们看到的不是这样的，马早就有了，而且那些故事把王亥说的好像他就会驯马，这也太贬低他了，其实王亥他有师父，学了能驯服天下各种奇珍异兽的本事，所以他能一眼看出黄帝骑的白马其实是麒麟。我们看到，本次人类文明历史中，黄帝所在的那个时期是最美好的，人与神互通，不仅各种生活必须品应有尽有，而且很多现代人无法想象的事物都有，很多神奇的制造工艺是通过人与南瞻部洲上面的生命配合着制造出来的，现代人根本无法想象。黄帝之后，中国的社会形式就一直往下滑，一代不如一代，最后来到今天这种社会形式。

贾齐骑着一匹黑马是王亥送给他的，然后王亥自己骑的是一种很奇怪的动物，历史上有说王亥驯服了“火畜”，我们就把他骑的这种动物称为“火畜”好了，因为这种动物我们在《山海经》也没找到。“火畜”是马的身形和马的蹄，皮是绿色的上面有麟甲，和《山海经》北山经说归山那里有一种叫hun的动物比较象，但和它那里描述的也有所不同，因为北山经说hun头长着羚羊角嘴巴象鸟，但王亥骑的“火畜”头部那里有红色鬃毛，头部长的是龙角嘴巴象龙嘴能吐火。

#  七印解封（34）：颌氏之子

2020/10/18
在今天内蒙兴安盟一带有一个氏族，被称为君火氏，黄帝他们离开调雨氏后就日夜兼程往这个君火氏所在地方去。这一天傍晚，眼看太阳要落山而又正下着雨，天气比较冷，一天又没遇到吃饭的地方，大家正是又冷又饿、人困马乏之时，见前面有个大宅子，于是大家策马来到门前敲开大门，说明来意。这家主人也很好客，就招呼这一大帮人马进去，又是喂马又是备饭，非常热情。

大家吃饱，闲聊起来，这家主人姓颌，就问黄帝他们从哪里来要往哪里去，黄帝就把自己从汉氏都城来想联系天下各处诸侯共同讨伐炎帝之事说了。颌氏膝下就只一个儿子，年纪轻轻，也正是踌躇满志之时，一听说打仗，就吵着求他父亲让他跟着黄帝他们走。黄帝看他父亲面露难色不太愿意，于是就想让他知难而退，就指着其中一名精兵对这位少年颌氏说打仗非同小可，你要去也可以，但必须要打败他。这个颌氏少年很高兴就说好啊我们比比。颌氏站着不动，让精兵冲他过来，精兵觉得就吓吓这个少年罢了，于是拿着武器很威武的样子慢慢地走向少年，眼看到少年跟前的时候，突然少年不见了，一扭头看到少年不知怎么就站在身后。精兵很吃惊，知道遇到高手了，就使出全部武艺攻向少年，但都是眼看要打着少年的时候，少年就总是在他身后站着。精兵只好认输，黄帝也很吃惊。黑者不服气了，上前和少年你来我往地比起来，最后两个人将将打个平手。黄帝很高兴，就对颌氏老人说您的儿子武艺高强，我很想把他带走，这时候少年母亲知道了，就从后院出来不同意，哭着说我就这么一个儿子，他走了我们怎么办。少年非要跟黄帝走，说自己从小学那么多武艺，现在正好能派上用场，否则当初还学这些有什么用。黄帝就对颌氏母亲说，您不如全家都搬到汉氏都城去吧。颌氏父母一想这样也好，于是黄帝就给他们画了到汉氏都城的地图，让他们先去那里，等自己带颌氏办完事后再和他们团聚。

颌氏一身书生打扮，衣服里面有十一把匕首，其中肩后两把，腰后两把，胸前两把，腰部两把，腿上两把，左臂上一把，骑棕色马。于是，黄帝身边又多了颌氏这位得力干将，我们看了一下，他的右肩膀上也有创世主的授记，也是当初随创世主从昆仑之顶下来的。

第二天，黄帝一行三十人离开颌氏又继续赶路，不一日就到了君火氏都城。君火氏很会用火，将士们的枪都带火，叫火枪，头盔也带火，是从一位姓李的修道人那里传下来的。君火氏君一听说要打炎帝，就毫不犹豫地同意了黄帝到汉氏都城会盟。今天君火氏都城那一带的地下还埋着一些他们用过的火枪。

离开了君火氏，黄帝他们就继续往北，来到了石氏都城，这个石字前面是有一个三点水的，就像少典两个字一样，因为我们从字典查不到这个带三点水的石字，所以我们就称之为石氏。石氏都城在内蒙呼伦湖一带，他们使长枪拿盾牌，颜色都是土那样，但那个长枪的枪尖那里能够飞出石块，盾牌那里能射出土弹。今天呼伦湖的湖底的土里还能找到他们当初用过的一些武器，因为这个石氏有个风俗，就是如果将士死了，他的尸体和他的武器就会被一起沉到这个湖中。这个石氏君他没有马上答应黄帝会盟的要求，而是让黄帝帮他们去山里找一伙强盗，因为这伙强盗会一种特别的隐藏术，一般人无法找到他们。因为这伙强盗头子手中有一颗宝石，石氏君很想得到这颗宝石。于是黄帝就带着黑者他们进山里找到了他们，并拿到宝石交给了石氏君。石氏君很高兴，就答应了黄帝的会盟要求。我们看了一下，原来历史上石氏曾经出过一位修道人，他会一种起死回生的道术，很复杂，其中就需要这个宝石作为其中一个关键的引子，到这一代石氏君也一直在收集这些东西。

离开石氏都城，黄帝他们就往西来到今天蒙古国乔色山一带，这里有一个很大的诸侯，有四座城，被称为辶井氏，在辶这个偏旁里面应该还有一个字，但我们却看不清，所以我们就叫它辶井氏。这个辶井氏君人不错，他不太想动干戈，所以开始的时候不想与黄帝会盟，但见黄帝锲而不舍地来劝说，于是这个辶井氏君就设下六大关希望黄帝知难而退，说如果六关都胜了，那么辶井氏就出兵与黄帝会盟。这六关分别是读书、驯兽、下棋、跑步、打靶、打架。

读书这一关要读的那本书不是当时东方文明的书，里面的文字就象今天人们知道的古希腊那样的字母文字。辶井氏君让人拿着这本书到黄帝他们跟前，问谁知道这本书讲的是什么内容。黄帝第一位师父精通天下所有事情，这种文字也都教过，于是黄帝就说我来，然后就拿起这本书边读边给大家说讲了什么什么。

驯兽驯的不是一般的兽，是一个可变大变小的兽，不是人类这个空间的，来自南瞻部洲地上面。辶井氏君让人把这只兽带到一个森林里放了，然后回来告诉黄帝他们这个兽的名字，让黄帝他们去山里找到它并带回来。这一关就非王亥莫属了，具体怎么找到怎么驯服的我们就掠过了，反正当这个兽看到王亥就变成了一个有今天三层楼高的那么一个怪兽，后来就被王亥驯服从森林里带回来。

下棋这一关黄帝派出了白氏将军，他们下的不是我们今天知道的那些棋，棋盘有两个房子大，中间用一个帘子隔开，两边棋盘一模一样，里面画的是各种山川河流城池之类的，以及将领守兵等，其中分布有一定的规律，中间有一个楚河汉界的分界线，两边分别选定一边，并自己选定首都所在。在帘子的两边站着两个人，用布把嘴巴封严了，负责看对方走法阵势，各自主将只能问这个看阵的某个地方是否有敌方将士，而且每一步棋只能问一次，这个看阵的只能回答是或者否。当如果这个地方双方将士同在，那么就有裁判根据兵力情况判断此地哪方胜，然后拿走败者将士以及胜方需要扣除的将士，胜方在这个地方被扣除多少将士败方不知情。大概就是这么一种棋，看着非常精彩，最后是白氏将军胜。今天辶井氏都城所在的地下的某个地方，还埋着一个用特殊材料打造的箱子，里面有一本书，是象一块块小木片那样组成的卷起来的书卷，里面就写了这种棋怎么下的方法。

跑步就是看谁先到终点，怎么跑都行。辶井氏君派出的将军脚下有一双特别的鞋子，穿上它就象飞一样，连八百里快马都跑不过他。可是他这次遇到的是颌氏。反正无论辶井氏将军怎么跑，最后都是颌氏在他前面。

打靶就是很远的地方有一个靶，看谁能够打中中心，如果都打中，就把靶移动到更远地方，反正就要分出胜负。辶井氏君派出的是神箭手百发百中，黄帝派出的是黑者拿飞镖。两人比了很长时间，靶都要看不见了，最后是黑者赢。

打架就是双方拿武器拼杀，黄帝派出的是贾齐，辶井氏派出的是他们最强的武士，使一杆枪和盾牌，这枪和盾牌都是特制的，即使是贾齐拿着的那把带火的神斧，也不能一下子把它砍破。相方你来我往打了很长时间，最后是辶井氏武士的盾牌被贾齐神斧砍破了只好认输。

黄帝他们六关全胜，辶井氏上下无不心服口服，于是同意派兵与黄帝在汉氏都城会盟。

#  七印解封（35）：痱农役使种子

2020/10/19
黄帝一行人离开了辶井氏之后就日夜兼程往汉氏都城方向走，沿途有不少小诸侯，也都一一愿意和黄帝会盟。其中有一个诸侯会发电，就是他们可以通过一种特别的方法从空中产生电，然后用这些电来做很多事情，例如照明，但那个照明器具不是今天的电灯，而是象西方人用来放蜡烛的摆件。还有一个诸侯也比较奇怪，他们喜欢吃骨头，但不是直接啃骨头，而是他们有一种方法把骨头做的就象肉那样。

这一路，黄帝的队伍中又增加了两位高手，其中一位叫痱农。在今天蒙古国巴安德尔格靠近内蒙古那里有一个小诸侯叫维氏，前面应该还有一个字，但我们看不清，所以就称为维氏。痱农是维氏都城外一个以种菜为生的菜农，长得比较瘦，一个天天干活的菜农那样，但不知怎么回事，这段时间他的菜一棵都卖不出去，家里什么都没有了。这一天他实在受不了，心想今天无论如何都要弄到钱，但今天他还是一棵菜都卖不出去。在回来路上，他生出了一个念头，就是要在前面山路那里要装一次强盗抢点钱。于是他来到一条山路那里，看前后没人来，就从袋子里拿出一些种子撒在路的两边布了一个荆棘阵，自己就爬上一棵大树等人过来。

此时，黄帝他们恰好经过这里，当他们进入荆棘阵的时候，这个痱农就做了一个动作，种子突然就从土里长出来变成密密麻麻的荆棘丛组成的一个罩，把黄帝他们都罩在里面。贾齐一看，抡起神斧对着这些荆棘一顿乱砍，但是他这边砍那边长，砍不完。痱农在树上见贾齐那个砍法，知道这些人都非等闲之辈，担心耗下去荆棘阵很快就会被破，于是赶紧在树上喊你们只要给我留下一些钱，我就放你们走。黄帝一听，原来是强盗拦路要钱，于是就让贾齐把神斧收了，说你在哪里啊我怎么给你钱。痱农就说你看前面荆棘从那里有一个洞，你们把钱从那里放到外面。黄帝一看，只见前面荆棘丛地面那里慢慢分开出一个小洞。于是黄帝就把一些钱放到外面。痱农在树上一看就说你们给的钱太多了，然后就说了一个数让黄帝按这个数给他。大家一听都不禁笑起来，怎么还有嫌钱多的强盗。黄帝就问你强盗都做了怎么就要这么点钱。痱农就说我也不是强盗，我就是太穷了想弄点钱，美美吃一顿，再买点生活必须品。黄帝就说那不如你跟我走吧，我以后管你饱饭。痱农就问你是谁，我凭什么要跟你走。黄帝就把自己四处说服诸侯征讨炎帝之事说了。痱农就说炎帝那么强大，你怎么可能打得过他？你们现在说服了多少个诸侯？如果没什么诸侯跟你，如果我跟了你岂不就要变成个饱死鬼，那我可不愿意。黄帝就把一路上有多少个诸侯愿意会盟的事情说了。痱农一听都这么多诸侯跟他了，于是就从树上下来，手里抓一把粉向荆棘阵一撒，那些荆棘丛又变成了种子。

我们看了一下，痱农他有一个秘传的方子，用这个方子可以提炼出两种粉末，一种粉末用来控制种子的生长，一种粉末用来控制种子的回收，然后只要把控制生长的粉末撒到种子上，那么痱农就可以役使种子的生长随心所欲，如果把回收种子的粉末洒在植物上，植物就会变成种子。

黄帝他们先和痱农在附近找了个地方吃个痛快，之后随着痱农回到他的农舍，让他收拾东西跟自己走。痱农的农舍就是几间破房子，可是他却跟黄帝说还要等几天，他要把房子卖了才能走。大家一听都笑起来，黄帝也笑着说那你就把房子卖给我吧，于是痱农就出了一个稍高一点的价格卖给黄帝。待黄帝把钱付清，这个痱农就跟着黄帝一起上路。

在忍者森林里有一位武氏，黑者是他唯一的朋友，他使一杆长枪，这杆枪不是一般的枪，是用特殊材料打造的，拿在手上很轻，但如果打到人身上却又很重，比用铁做的枪还重。自从黑者在偷盗黄帝神剑那次行动中失败后，武氏以为黑者已经死了，觉得生活也没什么意思，于是就离开了忍者森林到处走。这一天他来到了今天内蒙古苏尼特左旗那里的一座山，山不是很高，有很多树木，山路很窄，走到半山腰的时候前面的路就只能过一个人，然后在这个窄路前面转弯的地方下面是很深很陡的一个悬崖。他看到这个悬崖突然就想在这个地方打坐，于是他就把枪一插，插在悬崖边，然后纵身一跳就凌空飞起，单脚稳稳地立在枪柄那个尖上，另一条腿盘在膝盖上，双手打一个手印然后双手从两侧伸出象开放的花瓣掌心朝上象托着两个花瓶那样，然后把眼一闭就定在那里打坐。

黄帝他们在维氏都城说服了维氏君到汉氏都城会盟后沿路只要遇到的诸侯都被说服了，这一天他们正好经过武氏打坐的那条只容一个人过的窄路，大家都下了马，黄帝走第一牵着白马转个弯，一抬头就看到立在枪柄尖上闭目打坐的武氏，很奇怪，就对着武氏招招手，见武氏没反应，就惦着脚以免惊动武氏继续前行，然后是痱农，然后是颌氏，然后是黑者，黑者抬头一看这不是武氏吗，于是把武氏喊下来相见，大家都很高兴，于是武氏和黑者一起就成了黄帝身边的左右护卫。

我们看了一下，痱农和武氏右肩膀上也都有创世主授的印记，他们都是当初从昆仑之顶随创世主下世的。

得了武氏，黄帝他们又是日夜兼程，沿路又说服经过的诸侯，然后就回到了汉氏都城。此行，黄帝他们历时约大半年，共说服了神龙氏东北部约二十多个诸侯。

# 七印解封（36）：黄帝的理想

2020/10/23
黄帝他们回到汉氏都城休整了几日，然后就踏上了往西部说服神龙氏诸侯的征程。黄帝这次西行没有带白氏的二十四位精兵。汉氏君给了黄帝一张地图，上面标有吕梁山以东一带到陕西以南炎帝部族中对炎帝不满的那些诸侯所在，于是黄帝就写了一些劝说信，让黑者和颌氏两人带着地图从今天朔州市一带入河北境内到黄帝第三位师父一带，经过黄帝第二位师父所在的诸侯那里，然后沿着西安、宝鸡那一带把这些劝说信暗中放进这些诸侯的府中，然后往兰州方向的某个地方与黄帝会合。然后黄帝就带着白氏、王亥、贾齐、武氏、痱农五人从汉氏都城出发沿黄河往南穿过今天的宁夏然后到甘肃兰州一带和黑者颌氏会合，之后从那里往新疆方向一路说服那一带诸侯，然后从新疆大概和田玉那里进入西藏、然后经青海、甘肃、宁夏返回武氏都城。

这次西行，黄帝共说服了约三十多个诸侯，历时约大半年，共得到十位能人追随左右，他们是常先、顺辘、岐伯、王雨、大鸿、后溟、伶伦、龙氏三兄弟。我们看了一下，他们右肩膀上也都有创世主授的印记，都是当初从昆仑之顶随创世主下世的。

这一天黄帝一行八人来到河氏都城，河氏前面应该还有一个字，但因为我们看不清那个字，所以就称河氏了，见城门那里立着一个牌子，上面写的大意是我叫常先，很会盖房子，不仅可以盖一般房子还可以盖城。黄帝一看，虽然收费很贵，但自己需要一座城，于是就按照那个广告牌写的地址找上门来。常先穿绿色的衣服，戴一个黑色帽子，个子又矮又瘦。常先待客的地方居然有一个象今天房地产公司那样透明玻璃封装的房屋模型，不同的只是那个房子是古代的房屋结构，而那个象透明玻璃的东西不知道是什么东西造的。大家互通姓名后，常先就带黄帝他们进到一间地下室，里面空间很大，有很多机械，比如现在人们通过滚动履带把石头、砖等传送到远处的机器，或者把东西切割成什么形状的切具，那里很多都有，不同的是它们都是要靠人力手摇或其它方式驱动，而不是用电。黄帝看到这些机械知道今天遇到奇才，于是就说自己想要一座怎么的城，常先就拿着个算盘边听边算账，黄帝说完常先就报了个价，黄帝一听说太贵了，又来回还了一下价，然后王亥也出来还了一下价，最后还是太贵，于是黄帝就说自己要征讨炎帝，想给世人带来一个依礼而行大道行于天下的大德之世，不如你和我们一起去为天下做些事情吧，常先开始不愿意，后来两人就你来我往地说了很多话，大概意思是常先说自己就想多挣钱，而黄帝就说你要多挣德而不是多挣钱，最后常先被说服，就交代家人跟着黄帝走了。

顺辘会登天术，那时候黄帝他们要过一个万丈悬崖，武氏的马不知怎么突然受惊，毫无提防的武氏就一下子从马背上被摔下了万丈深渊，大家都想这下完了，正着急之时，只见从万丈悬崖底下出来一个人，书生打扮，青蓝色衣服，背上一把剑，双手托着武氏的腰，居然凌空，就象走楼梯一样一步一步地踩着虚空走上来，把武氏放回马上，大家看着无不羡慕。我们看到的这个什么都没有的空间在顺辘脚下就象地，他想怎么走就怎么走。黄帝向顺辘说明自己此次西行的目地，希望他可以加入，顺辘马上就答应了，说自己等的就是今天。

在黄帝随行的这些能人异士中，白氏将军年龄最大，差不多有五十岁。这一天他们正在一条山路上走，忽然白氏背部旧伤复发，疼得无法走动，坐在地上痛苦不已，大家也只能眼看着在旁边焦急无能为力。这时刚好有一位到山上采药的郎中路过，只见他样子有些胖，满身袋子，从小腿一直到肩膀、双臂，反正是有空挡的地方都绑着一个小袋子，里面装着各种不同种类的植物。他看到白氏痛苦的样子就走过来说让我来，然后大家也没见他望闻问切的，就只是一只手板着白氏的肩膀，另一只手一推白氏旧伤复发的那个地方，然后就说好了，白氏也感觉好了。大家想，今天遇到神医了，互通了姓名，原来叫岐伯。黄帝就力劝岐伯跟随自己征讨炎帝，可是岐伯就是不愿意，说这一带的百姓很需要他。黄帝就说现在天下大病，正是需要大医之时，如果您能够和我们一起把天下大病治了，人们重德向神，大德之世就会来到人间，您这个地方百姓的病自然就得到根除，否则你终其一生，也就是在这里反反复复地给人看病，而无法根除他们的病。岐伯一听也很有道理，就对黄帝说我先回家想一想，明天你们在某个时辰某个地方等我，如果在那个时辰我没出现，那你们也不用等我了。从历史我们也都知道了，黄帝等到了岐伯。

王雨一身横练力大无穷，是以非氏都城的一名普通百姓，但侠义心肠。当时黄帝一行来到以非氏都城，忽然就见一群飞贼在光天白日之下在房顶上一边飞檐走壁一边往地面射箭，然后地面那里有一个大汉一边健步如飞地追赶一边挥舞拳头击向那些飞来的箭，拳头所过箭头都被打破。黄帝他们一看这个人的力气也真是太大了。但毕竟是飞贼，王雨力再大也只能是眼看着他们跑的无影无踪。黄帝看着王雨这个大力，心想攻城时正是撞开城门的好手，于是就说明来意，想邀请王雨和自己一起攻打炎帝，但王雨一口回绝，说我们这里有十几个飞贼，乡亲们需要他的保护，除非你帮我们把这里的飞贼根除了。黄帝就说那等我见了你们的君侯后我再回来找你，然后黄帝一行就来见以非氏君说明来意，以非氏君就说我们这里飞贼横行，他们轻功太高一打就跑，如果你能帮我们把这些飞贼根除了，那我们就出兵会盟。于是黄帝就带着黑者、武氏、颌氏等擒拿了以非氏都城一带的所有飞贼。

大鸿当他穿文人衣服的时候样子很象诸葛亮，能文能武，但他的武艺更高强，可能是当时整个黄帝时代排第二的。那时候黄帝他们来到化氏都城，化字后面还有一个字，看不清，我们就称化氏了，他们看到城里有一擂台，上面一大堆财宝作为赏品，大鸿是擂主，那些上来挑战的，都被他随手一拳就打下擂台。黑者一看真是高手就跃跃欲试，颌氏就拦着他说我先来，黑者说不行，两个人在台上就争起来。大鸿说你们两个一起来，于是黑者和颌氏两人一起挑战大鸿，你来我往，颌氏身体瞬间移动的速度、黑者忍者之术在他拳头面前好像都太慢了，居然都没法躲开这个人的拳头。黄帝看着也技痒了，也上来和大鸿比。大鸿看着黄帝腰上神剑说你不能用你腰上那把剑，其它的任你选。黄帝的本事有些特别，他用手在自己的胸前一抹，然后就出来很多绿色雾把整个擂台罩住，然后就突然出现在大鸿身后想把大鸿击倒，但大鸿背后就象长了眼睛，在黄帝拳头即将打到身体的时候突然就抓住黄帝的拳头把黄帝整个人都扔到自己面前。黄帝一看就拿双剑使天兵剑法，大鸿就腰上挂一把宝剑不出鞘就把黄帝打败了。黄帝爱才深切，就在擂台上对大鸿说自己想征讨炎帝然后带领世人建立大德之世，希望大鸿能出来辅助。大鸿这个人不爱说话，平时也很少话，就说你这些道理我都懂，如果你们谁能在这个擂台上打败我，我就跟你。于是贾齐拿神斧跳上来，结果被大鸿点穴定在那里无法动弹。武氏上来，他把枪使得象千斤坠一样从上往下猛地向大鸿头顶砸下，大鸿用手臂就把武氏的枪隔开，武氏连退几步，枪差点脱手。王雨一身横练力大无穷，但是大鸿动作太快，王雨拳头根本挨不着他，可是大鸿拳头打到王雨身上就象打到钢板一样拳头都痛也是白打，于是大鸿就用点穴把王雨点在那里无法动弹。

能武就剩白氏了，但白氏站那里就不上擂台，觉得这人太厉害自己上也白搭。于是痱农上擂台，从衣服里掏出很多种子撒向大鸿，大鸿也不知道痱农想干什么，然后就见痱农说了一声“长”，大鸿周围的种子突然就长成很多藤条把大鸿整个人一层层地密密缠住，就象一个巨大的蚕茧。痱农哈哈大笑说你还不赶紧认输，话音刚落，突然就从茧里伸出一个拳头把痱农打落擂台。顺辘拿着一大捆铁棍踩着虚空蹬到大鸿上空，居高临下把那些铁棍一起用力插向大鸿，就象密密麻麻的棍雨，顺辘在空中正想这回你还不认输，就见一根铁棍从下而上一下就把顺辘从空中打落尘埃。常先不仅会造房子，他还会造各种机器各种暗器，他上来拿着一个象睡觉枕头那样的暗器机关向大鸿发射暗器，如果这个暗器打到身上，大鸿就会不断打哈欠。我们看到，大鸿他武艺高强的其中一个主要原因是他的速度太快，连颌氏那样的速度在他眼中都是慢的，常先的暗器对他是一点用都没有。大医都是用毒高手，只见岐伯上擂台双手飞快地从身上那些袋里掏出粉末撒向大鸿，大鸿连忙闭住呼吸和全身血脉，然后一拳就把岐伯打下擂台。最后王亥上来，在擂台上手舞足蹈，口中发出一种奇怪的声音，天上就飞来一大群有毒飞虫，大鸿一看，连忙从口中发出一种啸声，那些飞虫一听都吓得赶紧飞走了。白氏一看，只剩自己了，只好上擂台与大鸿相互见过礼，然后就自报门户说我是谁那里人氏，今日得与你切磋武艺真是三生有幸之类。大鸿一听连忙问你祖上是谁，原来住哪里，等等查家谱一样的问题。白氏就说自己父亲是谁、爷爷是谁、爷爷的爷爷又是谁，原来住什么地方等等。大鸿一听就说，我的族人是谁，来自那里，在什么时候我们族人欠了你们白氏的，我们一直想报这个恩。于是大鸿就跟着黄帝踏上了建立大德之世的征程。

中国古代神话传说中有说帝俊与羲和生了十个孩子都是太阳，其中九个被后羿张弓搭箭射了下来。我们看到后羿这个人在历史上是有的，箭术高超，可以说到了天上神箭的境界他射出的箭具有神箭威力，但后羿射日这个事情我们却没看到，可能不是真的。后羿后人经过很多代之后，来到黄帝这个时期，他的名字叫后溟，他也掌握了密传的神射术，但还没能到神箭的境界。后溟箭术超群，而他那些族人射术都一般，就是普通将士那个水平。这一天黄帝他们路过一个树林，树林的上空正好有一行大雁飞过，忽然就看见从树林里飞出一杆箭，这箭会拐弯，从领头的大雁那里穿过，从最后那只大雁那里出来，一群雁就这样被一箭射了。大家看着这箭术也太少有了，于是就往箭落下的地方寻去。双方互通了姓名，黄帝说明来意，后溟说我们族人不会什么手艺，就靠他这身箭术谋生了。黄帝觉得失去这样的神箭手也太可惜，于是就和后溟来到他们族人居住的村落。常先周围看了看说，我可以在这里给你们盖一个寨子，让你们族人安居乐业。于是常先就选了一个靠山的地方，大家就在常先的指导下动手盖寨子。我们看到这个寨子太好了，靠山而建，象三角形，如下图所示：



现代人讲生态循环，常先盖的这个寨子就是一个自给自足、自我循环的生态系统，房子的布局非常考究，阳光分布、进出规划、高低设计，等等，无不体现着人与自然的和谐。寨子有巨大的天井，下面有一块地可以用来种东西和养动物之类的，也可以练武等等，天井上有廊桥把两边的房子连起来，而在房顶靠山的那个地方有一个鱼塘，它和山上流下的水组成了一个循环系统。鱼塘的构造也非常讲究，使得那些山里的鱼都很喜欢往这里跑，所以族人根本不用养鱼，如果想吃鱼，就随手从塘里捞几条出来，吃也吃不完，生生息息循环往复。从鱼塘那里有石阶可以上山。住在里面的人一辈子不用出寨子也不会饿死，所以如果有强盗来，把那个大门一关就解决所有问题。后溟一看，没后顾之忧了，于是就背着一把神弓带着八盒神箭跟着黄帝离开了自己的族人。岐伯看着后溟的箭和弓就问您这是传说中的后羿所用的神弓神箭吧，后溟就说我们族人是这么说的，是祖先遗物，但我还达不到传说中先人那样的能力。我们看了一下，后溟的神射术是需要一种御气术辅助的，也就是当箭往前飞的时候，他通过御气术去控制箭周围的空气而改变箭的飞行轨迹。

中国神话故事有说伶伦用竹管钻研音律，开始阴阳不分还把黄帝的马惊吓了。其实我们看到的不是这样，伶伦的音乐水平可能是本次中国文明四千五百年以来最高的，后人应该没有能超过他的了。那时候黄帝他们正路过一座森林，忽然远远听到有美妙音乐飘来，那个美妙是从未有过的，于是大家就寻着音乐来到一个开阔的地方，那里有一棵树，看到一个年轻人坐在树上，象书生，相貌英俊，穿白色衣服，手拿着一把琴在弹奏，这把琴象琵琶但有六根弦。稀奇的是，这个开阔地上坐满了各种动物，不仅走兽，连那些飞禽也都在地上坐着。有腿的不管是飞禽还是走兽，全部象人一样坐在地上伸出两条腿，嘴巴发出各种声音居然与音律相配，身体随着音律摇摆，而没有腿的象蛇那样的爬虫就盘着身子伸着头在那里左摇右摆，如痴如醉。平时那些你吃我我吃它的天敌今天都坐在一起了，相安无事。黄帝他们看到这副奇妙景象，无不赞叹，大家也都被音乐感动双目微闭也象那些动物一样沉浸其中了。只见黄帝威严站立手大开大合如开疆辟土气吞山河，而大鸿威严站立却一动不动如泥塑木雕，岐伯则手舞足蹈跳起舞来，而剩下的其他人都全部躺在地上，各种躺姿，各种表情象在呼呼大睡但又做着各自的美梦。曲终鸟兽散，黄帝他们也才回过神来，顿觉心旷神怡身轻如燕，大家无不啧啧称奇。伶伦从树上下来与大家见过礼互通了姓名，黄帝说明自己此行目地，力邀伶伦一起同行。伶伦推脱，说自己一音乐人不懂什么打仗，黑者就说你可以在战场上弹奏一曲让炎帝的兵马全部睡觉，那大家都不用拼杀了。伶伦说自己的音乐之力有限，无法让炎帝兵马睡觉。黄帝说当今人人追求异能奇术，天下失德，德才不配，从而导致天下众多异能之士纷纷逞强斗能，盗贼横行。你的音律能让飞禽走兽自以为是人，让天敌坐一起而秋毫无犯，真乃大音。如果你能和我们同去平定炎帝，您的大音就能让天下人听到，天下德才就会兼备，人间将迎来大德之世。于是伶伦就跟着黄帝等人走上了辅助黄帝实现大德之世之路了。

龙氏三兄弟是孪生三兄弟，长的一模一样，其中一人拿关羽那样的青龙偃月刀，一人拿张飞那样的丈八长矛，一人拿吕布那样的方天画戟。那时黄帝他们看到山路上约有几十个武艺高强的强盗拦路抢劫，正想前去帮忙，却见从道路的另一头冲来龙氏三兄弟几下就把几十多个强盗全打败了。黄帝一看龙氏三兄弟武艺也真是高强，于是就前去邀请，龙氏三兄弟说除非你们有人能打赢我们，于是黄帝用手一指大鸿说你们和他切磋一下。大鸿拿着一把剑和龙氏三兄弟比起来，比了一会，大家就不打了。大鸿说我们一时也分不出胜负，但按照这样打下去，最后龙氏三兄弟应该会赢我一招，龙氏三兄弟说你们这里一个人就这么高强，我们愿意跟你。

#  七印解封（37）：右肩膀印记

2020/10/26
黄帝结束西部之行回到汉氏都城，共说服了约六十个诸侯，历时约一年两个月，除去白氏，共得到十六位能人异士跟随左右辅助，他们是黑者、王亥、贾齐、颌氏、痱农、武氏、常先、顺辘、岐伯、王雨、大鸿、后溟、伶伦、龙氏三兄弟。汉氏君信守诺言，把整个汉氏交给了黄帝。于是黄帝以汉氏为基础，城墙上插五色德字大旗，命常先带人在汉氏都城外兴建简易行宫以迎诸侯，在行宫外建一个议事厅，议事厅外建一个祭台，祭台外建演武场，演武场建一点将台，演武场外建各诸侯大军驻军之地。然后命岐伯统筹医药，命王亥统筹军马畜牧，命痱农统筹粮草，命伶伦制诸侯会盟、行军、驻军、作战之礼乐。中国人后来都称汉族，不仅仅是因为刘邦，这个渊源可以追溯到黄帝统一中国是以汉氏为基础的。

常先对黄帝说我们现在大兴土木，然后这许多诸侯前来会盟，还有后面征讨炎帝等等之事，必须一个统管调度钱粮之人。黄帝说我这段时间也一直忧心此事，但又找不到精通此道之人。常先就说我河氏都城里有一个朋友叫隶首，过目不忘精通此道，那时我那些工程都会请他来帮忙，他只看一遍，就能把所有事情分配的井井有条绝无差错，我觉得他非常合适。黄帝说太好了，便让常先写了一封信让人把隶首从河氏都城接过来。隶首模样和常先差不多，也是又矮又瘦。

中国历史故事说黄帝通过梦的启示找到风后，我们看到这是真的。怎么制定讨伐炎帝的战略、怎么练兵布阵等等，黄帝知道自己还缺一位有经天纬地、定国安邦之能的军师，日日忧心。这天黄帝梦见从以非氏都城那里刮起一股大风把天下的尘垢吹去了。黄帝醒来知道这是上天启示，要找的军师应该在以非氏都城姓风名后。以非氏都城大概在今天新疆巴音郭楞一带，黄帝想亲自前去相请，但路途遥远，诸侯正在前来，自己不能离开太久。王亥说这好办，你的白马是麒麟，一天就来回。黄帝说它现在是白马，王亥说我可以让它恢复麒麟本相。岐伯也想一起去，因为那个地方有一种草，现在正是采摘的时候。于是黄帝就带着岐伯骑着麒麟一会就到了以非氏都城，这时候以非氏君已经带着将士正在前去汉氏都城与黄帝会盟的路上，黄帝见到代理守城的官员说明来意，那位官员说这个容易，我们以非氏有个规矩，就是男子成年后都要写一份自己有什么才能的书卷交到我们这里以备不时之需。不一会就有一位官员拿着一个书卷出来，黄帝展开一看，里面写着风后精通兵法布阵等文字，黄帝说肯定是他了。风后长的象庞统，这时候他还是以非氏一个烧陶卖碗为生之人，也教几个学生布阵兵法之道。

不久，各大诸侯都基本到齐，黄帝带风后和隶首连日与众诸侯在议事厅开会，以了解各诸侯军力、粮草等情况，商讨军队建制、征讨炎帝战略、粮草供应等重大事宜。议事厅是一个圆形建筑，房顶不是往上凸起的，而是往下凹，中间开一个圆形的口，这个口下面是盟主之位，围绕着盟主之位有很多座位排成两个圆，它们是众诸侯之位。

中国历史说黄帝通过梦的启示找到力牧，我们看到这个故事也是真的。这天黄帝做了一个梦，梦见广阔草原里有很多蒙古包，一人手持千钧之弩在那里牧羊。黄帝醒来知道这是上天启示，广阔草原里有很多蒙古包意指前来会盟的众诸侯，要找的人叫力牧。于是黄帝就问众诸侯是否有叫力牧的这么一位将军，其中一个小诸侯说我这里有，黄帝就带大鸿去找力牧，他长的有些象吕布，头盔上带两把刀，左边腰上挂一把刀，右边腰上挂一把剑，背上插着三把短枪和一把短画戟，手拿青龙偃月刀。黄帝让大鸿和力牧两人用拳头比试了一番，打了许久分不出胜负，两人应算并列天下第二了。

一切就绪，于是黄帝带众诸侯来到祭台，祭台上高高插着五色德字大旗迎风飘扬。祭告了天地，黄帝当众宣读告天下书，诠释德之要义，列炎帝所行不德之事，昭示天下讨伐炎帝以建立大德之世。然后黄帝带领众诸侯拜风后为军师，统领众诸侯所有将士，号云师。风后接过符印来到点将台，把演武场上众诸侯军队按其氏族先天属性、后天所得，按阴阳五行相生相克之理对众诸侯军队进行搭配，划分为左、中、右三军。其中左军以大鸿为大将军举红色麒麟旗，中军由黄帝主掌举黄色龙旗白氏代行大将军之职，右军以力牧为大将军举蓝色麒麟旗。

我们看到，所有在这里会盟的诸侯、将军、士兵等，他们右肩膀上都有一个创世主授的印记，这个印记的样子和法轮大法里的法轮图形一模一样。一眼望去，整个演武场上都是法轮。

#  七印解封（38）：炎帝

2020/11/07
烈山氏在四千七百年前人神共通刚开始的时候也是属于被道家修炼文化影响很深的氏族，整体道德也较高，但在后来发展中，却与神龙氏走了一条相反的路。神龙氏一开始虽然很强大，但他们对诸侯并没有采取很多强制性的约束，更多的是以德服之，诸侯更多的是以信义从之。但烈山氏到黄帝时期，炎帝就建立了一个类似于中央集权的这么一种制度，从河北到江浙，从湖广到四川陕西部分，都在他的强力统治下。

炎帝手下有一名谋士叫“连井氏”，连和井之间还有一个字，但我们看不清，所以就叫他“连井氏”了，他足智多谋，整个炎帝建立的中央集权制度大部分都出于他。主要有三大策略，第一是收兵权：神龙氏诸侯有独立的兵权，但是在炎帝这里，他向所有诸侯都派驻了监军，相当于诸侯的兵权都被收到中央；第二是以金钱和武力立国：炎帝推行的不是以德立国，而是金钱和武力，例如如果一个强盗本领高强占了一大片地盘，那么这个强盗可以用钱向炎帝买一个诸侯来做把自己洗白，而不管他之前的罪恶滔天；第三是建立监国制度，把整个烈山氏诸侯划分为东南西北四部，每部设大监，均为武力高强的忠心之士。

东部大监叫“迨氏”，“迨”字前面应该还有两个字，但我们看不清，就叫“迨氏”了，他使一条枪；南部大监就是历史上有名的蚩尤兄弟了；西部大监叫“孙氏”，“孙”字后面还有一个字，但我们看不清，所以就叫“孙氏”了，他们是四兄弟，全部使剑；北部大监叫“形氏”，“形”字前面还有一个字，但我们看不清，所以就叫“形氏”了，他有四只手，每只手上拿一把斧头。

这四部大监每一个人的武功都非常高强，而炎帝身边还有一个贴身护卫，他使一把关羽那样的青龙偃月刀，应该是天下第一了。之前我们以为大鸿和力牧是天下第一，但写到这里我们才看到原来炎帝身边还有一位和他形影不离的贴身护卫，后面他曾经和力牧有一战，大概三十招之内就把力牧打败了。

炎帝不象龙帝那样贪于享乐，可以说为了烈山氏统一中国的野心不辞劳苦。他专门在四部里建立了四座宫殿，东部称为春宫，在浙江一带；南部为夏宫，在湖南境内；西部称秋宫，大概在今天陕西西安一带，北部称冬宫，在河北境内。炎帝一年四季都奔波在这四个地方：春天他到春宫办公，夏天到夏宫办公，秋天到秋宫办公，冬天在冬宫办公。

黄帝与神龙氏诸侯在汉氏都城会盟大练兵马准备征讨炎帝的消息早就传到了炎帝耳中，就和“连井氏”商量，都认为这是一举歼灭神龙氏各诸侯精锐部队征服神龙氏统一中国的大好时机。因为马上要到秋天了，炎帝正在夏宫，就决定在秋天的时候从秋宫那里出发与黄帝决战，于是，炎帝命冬宫大监和春宫大监带领所管辖诸侯精锐部队前往秋宫集结，自己就带着蚩尤等南部诸侯精锐将士开赴秋宫。

#  七印解封（39）：秋宫之战

2020/11/15
《五帝本纪》说黄帝与炎帝战於阪泉之野，三战，然後得其志。我们看到的和司马迁写的有些不同，皇帝与炎帝有很多场大战，而且很多大战发生在不同的地方，并非如世人理解的阪泉之野。炎帝从秋宫出发，黄帝从汉氏都城出发，两军在今天陕西省吴起县附近一带开阔地带相遇，打了很多场仗，为了行文方便，我们把这里发生的战争统称为秋宫之战。

秋宫之战的第一仗黄帝这边摆出的是方形战阵，阵的前面是黄帝骑麒麟带着一众将士，阵的后面有一个木头搭起的指挥塔，风后在上面执掌各色令旗，而阵中各个氏族军队都有专门观看风后令旗的传令兵。黄帝战阵的最前面一排是石氏将士组成的盾牌军，所以黄帝他们摆出的是一个防守战阵。而炎帝只命西部大监带他们的部将摆出进攻战阵，第一队列将士都是弓箭手，队列是弧形的，弧的凹面朝向黄帝他们，就象要把黄帝战阵吞没一般，然后弧形弓箭队列后面是方形的战阵，各种冲锋军种依次排列，战阵的前面是西部大监孙氏四兄弟带着他们的众大将，后面并没有执掌令旗的指挥塔。炎帝和各诸侯大军在后面压阵，炎帝自己则在一处高地那里，上面搭了一个看台，他就坐在宝座上和众大将悠然观看，明显就没把黄帝他们放眼里。

在炎帝战阵那个地方，痱农提前布了一个荆棘阵，当双方一开战，他就发动了荆棘阵，那些西部大监很多军士一下就被荆棘托到半空，或者直接被荆棘里的利刺扎死或扎伤，于是西部大监摆出的战阵就大乱了。炎帝在看台上看着很生气，立即让身边一位会御火术的能人施展法术，只见从虚空中出来火都把荆棘点燃破了荆棘阵，同时炎帝让身边一位会驾驭闪电的能人施展法术，只见天空阴云密布，很多闪电突然出现击向黄帝他们，众将士纷纷四散逃命，风后见此赶紧挥舞撤退令旗，黄帝与众将士败回大营。

黄帝挂了免战牌与众将商议，常先对黄帝说他有破闪电的方法，于是黄帝让众将士按常先说的准备，然后去了免战牌约定明日会战，这次黄帝摆出的是进攻阵，最外面队列是执长枪的将士组成的三尖叉一样的队形，阵的最后面是后溟带着弓箭手组成的队列，而炎帝他们还是昨天一样的阵形。当双方一开战，后溟就带着弓箭手向敌方射箭，但他们都故意射不到炎帝的战阵那里，只是射到阵前。炎帝他们看着哈哈大笑，都以为黄帝他们不堪一击，于是又象昨天一样释放闪电，黄帝众将士一见天上闪电，手中有长枪的立即把长枪或者长刀之类的插到地上枪尖朝上，没长枪的都用事先准备好的一根长铁插到地上，就象今天人们知道的避雷针。然后风后把令旗一挥，只见黄帝的部队迅速出击，一下就把没有提防的西部大军击溃，然后风后令旗又一挥，黄帝众将士就立即退回大营，并没有乘胜追击。

双方稍作休整，第三仗就开始了，这次炎帝不敢小觑黄帝，亲自带着众将士来到阵前与黄帝对阵。黄帝摆下的仍然是方形防守阵，炎帝仍然是西部大监将士摆出的进攻队形。黄帝和炎帝一番唇枪舌箭之后，两人回归本阵，然后黄帝这边派出武氏出战，炎帝那边也派出一员大将迎战。只见武氏下马提枪站在地上，当炎帝那位战将策马摇枪冲来的时候，武氏手中那杆枪突然脱手快如闪电飞向敌人，一下就正中敌人胸部，然后武氏飞身上马从敌人尸体上取回长枪。炎帝旁边气恼了一员大将，立即冲出来大战武氏，十几会合武氏败回阵中，力牧一看立即手提青龙偃月刀上阵击败炎帝大将，然后炎帝那边车轮战一般有很多战将出来与力牧打，最后都被力牧打败。炎帝贴身护卫说此人武艺高强，普天之下估计只有我能胜，于是提青龙偃月刀出阵大战力牧，两人各施展平生所学，两把刀你来我往激战，大概三十招之内，力牧就勒转马头败下阵来，炎帝护卫紧追不舍。炎帝一看，立即命众将士乘胜杀向黄帝，黄帝这边阵门闪开把力牧让进阵中，然后阵门关闭石氏盾牌军那个盾牌发出很多很硬的土块把冲上来的炎帝将士打下一大片，然后身后骸氏将士手臂的骨头会突然暴长，手拿着武器对着冲上来的炎帝将士一轮乱砍，而后溟带着众弓箭手百发百中。

西部大监的军队几轮进攻都被这样打退，于是炎帝就派蚩尤兄弟，大概有十位，带着他们的一队甲兵手提大斧换下西部大监的冲锋队。历史上说蚩尤兄弟铜头铁臂刀枪不入，我们看到他们是一身横练，一般的刀枪伤不了，而且他们全身都穿着一种特别的护甲，这些护甲很坚硬，寻常的兵器都伤不了它，水火也奈何不了它。一身横练加上护甲，确实如铜头铁臂一般。蚩尤兄弟带着甲兵冲向黄帝他们，风后一看连忙摇动手中旗帜，石氏和骸氏以及后面的君火氏等部族将士看到令旗，连忙从阵的两边一队队地撤出，最后露出一只很特别的部队。《五帝本纪》说黄帝“教熊罴貔貅貙虎”和炎帝打仗，我们看到的这只部队并非真的是熊、老虎、貔貅之类的猛兽，而是在与黄帝会盟的氏族中，有几个氏族的将士身体都特别强大，两个胳膊象大腿一样粗，真是长得虎背熊腰那个样，而且那个相貌虽然是人的五官，但模样看着确实有几分猛兽一般，特别在作战的时候他们脸部还专门画上图腾，双手都拿着开山大斧大锤等重兵器，猛一看也很恐怖。蚩尤那些甲兵一阵猛冲突然看到前面这一大队猛兽一般的大军没有提防，被吓破了胆一般就被黄帝的熊罴之军打败了。

炎帝挂了免战牌，派快马去南部大监把蚩尤一半的兄弟以及精锐甲兵调来破黄帝的熊罴之军。当蚩尤兄弟到来后，炎帝就去了免战牌，双方开始了第四仗。这一仗，蚩尤几十个兄弟当头冲锋陷阵，后面甲兵勇猛跟随，如入无人之境，把黄帝的熊罴之军击溃了，风后一见赶紧摇动令旗收兵，高挂免战牌。

黄帝营中摆着一件从战场上得到的蚩尤护甲，众诸侯一个个轮流上前，看是否知道这种护甲的破解之法。都看完了，其中有一位诸侯说知道这种护甲的大概成分，是由特别的铁、土、花为主料，再加其它很多辅料，然后用一种特别的水熬制而成，这些物料很多都不在人世间，需要到南瞻部洲上寻找。黄帝说既然知道了它的成分，那么不知有谁会破解之法？王亥说他知道怎么破，但需要南瞻部洲上面一种动物的血。于是黄帝命王亥和那位知道护甲成分的诸侯带一队特别精锐的将士去南瞻部洲取血。我们看了一下，这种动物有些象人间这里的猪。然后黄帝等人又继续商议怎么用这些血来破蚩尤甲兵，最后大家定下一个策略，命常先带人赶制一批战车。

#  七印解封（41）：黄帝游历天下

历史有说黄帝在涿鹿之战击溃炎帝后，就在泰山之巅会合天下诸部落，并举行了隆重的封禅仪式以祭天地。我们看到当时黄帝是会合了天下诸侯氏族到泰山，但目地不是为了祭天地，而是在那里与众诸侯氏族定下德治之约。当时前来订约的诸侯差不多有两百个，黄帝为天下之主，在泰山上制定了施行德治的法度，核心就是诸侯以厚德居之，百官则需德才兼备，并重点制定了怎么衡量德的一些方法以及与德教化相关的礼乐。

我们知道中国古代有尧帝为了考察舜的德行，先是“声闻”，就是周围人们怎么评价他的，然后把自己的两个女儿嫁给他以考察为人之道，然后又使舜担任各种公职以考察他的才，然后使他接待宾客以考察他的礼，然后又使舜进入大泽以考察他的命。那时候黄帝制定的怎么衡量一个人德行的总的原则和尧帝考察舜帝的原则是差不多的，就是用很长时间花很多精力去考察确认，例如有时候故意让人办成乞丐在这个被考察之人路过的地方，看他怎么对待乞丐等等，看这个被考察人如何应对。诸侯君主起着道德垂范的作用，官员则起着以身作则的作用，所以怎么考察诸侯与官员的德行就成了国家的头等大事。

当整个施行德治的法度确立后，黄帝就命人在泰山之顶立了一块石碑，上面刻着德治之法以及所有诸侯的名字，黄帝就领着众诸侯在石碑前对天盟誓立约，行德治之法，违约者将遭天地责罚，德不配者将被格。有历史故事说黄帝在泰山时，曾制作名为《清角》的乐曲，我们看到当时对天盟誓立约之礼是有乐曲，但不是黄帝写的。伶伦在黄帝实施德治方面是一个非常关键的人物。与德相配的是礼，而在一些重大事项上的礼都有乐相配，为此伶伦都相应的制作了与之对应的乐，使得人们在那个礼乐中真的能感受到那个场而身心得以清洗，例如在诸侯朝见黄帝之礼中伶伦就制作了叫《鸣起》的乐，又例如在泰山上黄帝领众诸侯向天盟誓时，伶伦也专门为这种礼编写了乐曲，后人称之为《清角》。我们从当时演奏场景来看觉得叫《清角》这个名还是有道理的，因为这个乐有些象基督教唱诗班，伶伦站在那里，一边弹琴一边唱歌，然后他的前面有一群人依次按特定方位站立，在那里和着伶伦的琴音和歌在咏叹。整个这个乐是怎么样的，修炼人听不到具体的，只能偶尔听到些吟叹。当黄帝他们正式开始行盟誓之礼的时候，伶伦的琴、歌声和吟叹之乐就响起，然后天上突然就祥云瑞气，天龙凤凰飞来盘旋，当时来到泰山的人都得以亲眼目堵这一天人合一的神圣。

地球上的国家，只要来到海边，麒麟就都会止步过不了海，所以黄帝的足迹大概就是在亚洲欧洲非洲这些地方。黄帝每去到一个民族，他就会到这里了解当地的风土人情，也研究这里的动物植物，也都会找这里的头领交谈，对他们描述中国德治的胜况和好处。所有这些民族，与当时的中国比起来，就象一个天一个地一般，房子大部分都是草和泥土之类砌的，衣服也很粗糙。看来似乎在大洪水灭世后，他们那里留下来的人一代两代之后很快就把上一个历史时期的那些制造工艺都忘了，然后人们就只好从原始社会出来那样从新来。大多数这些民族的语言黄帝第一位师父都教过他，但也有一些民族的语言没教过，这时候黄帝和他们沟通就比较困难。

黄帝不仅走遍亚洲欧洲非洲，而且也去南瞻部洲上面，也到处了解上面的风土人情，研究动物植物，也会找当地的国王交谈学习。和南瞻部洲上面相比，中国就显得粗糙了。麒麟在南瞻部洲上就能过海，所以在南瞻部洲上黄帝不仅游历了中国对应的那块地，他甚至越过了地球海去了更遥远的地方。历史上有记载说黄帝发明了很多东西，我们看不少是真的，但并非是黄帝发明出来的，而是他在南瞻部洲上看到一些好东西，他觉得如果人间有那个物质能造出来，他就会带一些回中国让人们按照这个来造。比如我们看到黄帝来到南瞻部洲，看到一座城，那个墙他觉得很好，就和随行人员一起研究，可是研究后发现造这个墙的物质人间这里没有，只好惋惜了。比如茶具，黄帝时期人们已经喝茶了，但都是用碗喝，后来黄帝来到南瞻部洲拜会一位王时看到那个茶具真是精致漂亮，于是就带了一些回到人间，让人们照样烧制茶具。

世人对黄帝的理解太过狭窄。黄帝约二十一岁艺成，一年兩個月说服众诸侯会盟，用时两年多战胜炎帝，约二十五岁统一中国。黄帝的胸怀超越了人能够想象的，黄帝的远见也不是世人能够明白的，他的魄力气概更是无与伦比，远远超越了创世以来人类历史上曾出现过的那些英雄人物、帝王将相。

南瞻部洲地球海中央这块与中国对应的地，为行文方便，我们以后都称其为中土，当上一次人类文明遭大洪水毁灭后，中土也同样在劫难逃，然后那里也和人间中国一样开启了一个新的历史时期。但当黄帝统一中国时，中土却还是诸侯割据，各自为政，并没有一统。黄帝第一位师父博古通今，不仅精通人类的历史地理万事万物，也精通南瞻部洲上的历史地理万事万物，自小便给黄帝打下了立志要为后世神传文化的开创奠定厚实基础的志向。黄帝游历天下也不仅仅是为了解风土人情，他其中一个非常重要的目地是要观测那些国家是否会破坏自己在中国奠定的神传文化基础。黄帝发现中土有些诸侯象神龙氏诸侯那样，也有不少诸侯国观念上与炎帝比较象，显然是败坏了。人间的中国有很多强盗出没，南瞻部洲中土则是妖魔鬼怪经常会碰到。那些和炎帝观念相像的诸侯对于黄帝所谈论的或者没有兴趣，或者嘲笑，更甚者一听黄帝谈论道德就直接送客了，或者干脆不见。在那些败坏诸侯中，有大概五个势力比较强大，其中一个最强大的就是世人知道的刑天了，他也是其中野心最大的，不仅想统一中土，也想来到地球称霸。至于被砍头的炎帝北部大监“形氏”，他并非刑天，他来自于南瞻部洲上，把他尸首带回复活的那位修道人和他父亲是很好的朋友，是受他父亲之托的所为。“形氏”复活后，他之前在人间的记忆全部被抹去了，就跟着这位修道人，好像炎帝这段历史和他没有一点关系一般。

黄帝统一中国一年后就开始游历南瞻部洲中土，花了约两年时间、多次往返，和众随行人员详细考察后，便下定决心出兵中土，讨伐上面那些道德败坏的诸侯，以保中国未来神传文化的开创。事关神传文化，事关人类未来，事关众生得救，黄帝马上行动，在他大约二十八岁的时候，便从众诸侯中征调了有特殊才能、能征善战的将士组成征讨中土天兵，黄帝亲自把第三位师父传授的天兵天将相关武艺教授给大家，以在南瞻部洲打仗之用。当一切都准备妥当，为了守护中国刚刚开创的德治之世，黄帝便命风后代理日常事务，命力牧、白氏、后溟、常先、龙氏三兄弟等与众诸侯协助留守，自己便带着中土天兵，大鸿、痱农、王亥、黑者、武氏等跟随左右，开始了神话般的中土之战。黄帝在中土之战共得了南瞻部洲五位高人追随左右，其中两位就是中国历史上说的女魃和应龙。

蚩尤八十一兄弟在黄帝与炎帝最后一战中只剩下约二十位活着，其他全部战死。开始他们大部分都被囚禁，后来都逃脱了，去到今天俄罗斯靠近黑龙江的一个地方，他们的护甲就是在那里打造出来的。这个地方住着一群修练人，他们的修练方法很奇怪，不用打坐，也不用做什么特殊的动作，就是整天在脑袋里默默地背诵一些类似于咒语之类的东西，坐着也背，走路也背，反正只要有时间就在那里背。蚩尤兄弟来到这里后，他们就把护甲改良了，然后又帮助蚩尤联络各种人才、收拢兵马，要回中国与黄帝再战。当黄帝中土之战正酣之时，蚩尤兄弟就出兵，一路势如破竹。黄帝这边知道了消息，但也没怎么担心，他相信风后等肯定能够打败蚩尤。风后他们果然没有让黄帝失望，击败了蚩尤大军。这时候，南瞻部洲上虽然黄帝还没打到刑天那里，但刑天知道与黄帝免不了一战，于是便乘机来到中国，帮助蚩尤重新训练军队，然后联合蚩尤一起打败了风后他们。黄帝获悉了这个消息后，便立即带兵回到中国与刑天蚩尤之军大战，最后诛杀了刑天。可是刑天本来是一个修练人，有些手段，他就把自己的身体复活了，就象历史上描述的那样，双乳为眼肚脐为口，逃回到南瞻部洲中土那里。黄帝知道了这个消息之后，又立即挥军回到中土征讨刑天。

当彻底诛杀刑天后，中土那些道德败坏的诸侯已经被征讨得七七八八了，于是黄帝就想着选一个人来担任南瞻部洲中土之帝以保护中土的德治，这样自己就可以放心地回到中国。中土的那些诸侯就推荐了一些人，其中就包括我们在《大红龙的来源》提到的中国皇。当黄帝开始讨伐南瞻部洲那些败坏诸侯之前，中国皇的师父早就洞悉一切，便先让中国皇下山了，去中土边远的一些地方，那里有些海岛，其中一些海岛对应到地球上的台湾，上面也有人住，让他在那里起兵攻打中土边远一些败坏了的诸侯，这些诸侯一般势力都比较弱。于是中国皇就按照师父的吩咐，全身白色盔甲座下金麒麟带红色火手拿一把亮银枪，枪头那里左右两边又有两把弯刀，来到中土边远之地，一边联合当地道德较好的村镇之类，一边征讨那些败坏的，慢慢地也在中土边远的地方建立了一个德治之国，远近闻名。黄帝也早就听闻了中国皇的事情，于是就对中国皇的德和才进行了一番仔细考察。中国皇不负众望，黄帝很满意，于是就把中国皇召到中土都城，让中国皇称帝。

从黄帝出征中土，到中国皇称帝，历时约四年。其中神话般的历史，如果需要细说我们后面再继续，这里先大概交代一下。
